# 第一章：人像摄影基础入门

## 1.1 什么是好的人像照片 — 判断标准

一张好的人像照片，不是器材参数的堆砌，而是技术与审美的结合。在你按下快门之前，先搞清楚"好"的标准是什么。

### 五个核心维度

**第一，情绪传达。** 人像摄影的核心是"人"。被摄者的情绪、性格、故事感，是照片的灵魂。一张技术上完美但面无表情的照片，远不如一张对焦略偏但情绪饱满的瞬间。摄影师 Peter Hurley 在他的 "The Headshot" 系列教学中反复强调：被摄者的"眼神"决定了照片的生死线 [1]。你需要的不是更大的光圈，而是让被摄者放松的沟通能力。

**第二，光线质量。** 光线是摄影的物理基础，也是人像摄影最关键的变量。好的光线能塑造面部立体感、柔化皮肤瑕疵、营造氛围。硬光（直射阳光、小面积光源）产生锐利的阴影和高反差，适合表现力量感和戏剧性；柔光（阴天、大型柔光箱、反光板漫射）产生平滑的过渡和低反差，是大多数美感和商业人像的首选。大师 Joe McNally 说过："光线的质量比数量重要一百倍。" [2]

**第三，清晰焦点。** 人像照片的焦点必须准确落在眼睛上——至少是离镜头更近的那只眼睛。这是人像摄影的铁律，几乎没有例外。国家地理摄影师 Joe McNally 和肖像大师 Platon 都遵循这个原则 [3]。眼睛是观看者最先注意到的区域，如果眼睛模糊，整张照片的冲击力直接归零。

**第四，背景控制。** 背景的干净程度直接影响人像的视觉质量。杂乱的背景会分散注意力，让被摄者淹没在环境中。背景控制的方式包括：大光圈虚化、选择简洁背景、调整拍摄角度、后期裁切。Scott Kelby 在他的系列教材中反复强调："如果背景里有比你拍摄对象更吸引眼球的东西，你就失败了。" [4]

**第五，后期处理适度。** 后期是人像摄影不可分割的环节，但过度的后期是灾难。皮肤磨皮过度变成塑料感、肤色偏移不自然、瘦脸液化变形——这些都是初学者常见的错误。好的后期应该是"看不出后期"。Syl Arena 在他的灯光教材中指出，后期处理的目的是"让照片看起来像你记忆中的那个场景，而不是你相机拍到的那个" [5]。

### 好照片 vs 好人像照片的区别

一张"好照片"可以是任何主题的——风光、建筑、街拍、纪实。但"好人像照片"有额外的约束：它必须让观看者的注意力首先且持续地停留在人身上。任何构图元素、光线处理、后期手法，都必须服务于这个目的。

实用判断清单：
- 眼睛是否清晰锐利？
- 背景是否干净或至少不抢戏？
- 光线是否塑造了面部立体感？
- 被摄者的情绪/表情是否自然？
- 色彩/黑白处理是否统一？

如果以上五条有三条达标，这就是一张合格的人像照片。五条全达标，这是一张优秀的人像照片。

## 1.2 相机与镜头选择

### 全画幅 vs 半画幅 vs 手机

**全画幅传感器**（约 36×24mm）是人像摄影的黄金标准。物理面积是 APS-C 半画幅的 2.3 倍以上，这意味着：
- **更好的暗光表现**：同等像素下，单个像素面积更大，进光量更多，高 ISO 噪点更低。实际差距大约 1-1.5 档 [6]。
- **更浅的自然景深**：同样焦距和光圈下，全画幅的虚化效果更明显。这是很多人选择全画幅的第一理由。
- **更宽的视角**：50mm 镜头在全画幅上就是 50mm 视角，而在 APS-C 上等效约 75mm（佳能 1.6x 裁切系数）。

全画幅推荐机身（2024-2025 年份，按价格区间）：
- 入门：索尼 A7III（约 ¥8000 二手）、尼康 Z5
- 中端：索尼 A7IV、佳能 R6 Mark II、尼康 Z6III
- 高端：索尼 A7R V、佳能 R5 Mark II、尼康 Z8

**APS-C 半画幅**并不是不能用。对于预算有限或刚开始学人像的摄影师，APS-C 的性价比极高。关键是要理解等效焦距的概念：一只 56mm f/1.4 的 APS-C 镜头（富士 X 卡口经典人像镜），在全画幅上的等效视角约 85mm，虚化效果接近全画幅 85mm f/2.0 的水平 [7]。这已经足够拍出很好看的人像了。

半画幅推荐：
- 富士 X-T5 + XF 56mm f/1.2（色彩科学优秀，直出即可用）
- 索尼 A6700 + Sigma 56mm f/1.4 DC DN（性价比极高）
- 佳能 R7 + RF-S 35mm f/1.8

**手机摄影**在 2025 年已经不是玩笑。iPhone 15 Pro/Pro Max 和三星 S24 Ultra 的计算摄影能力，在日常人像场景中已经能产出令人满意的结果。手机的人像模式通过多镜头深度信息 + 算法虚化，效果越来越自然。但手机的根本限制在于：
- 传感器太小，暗光质量远不如相机
- 光学焦段受限（主摄通常等效 23-26mm，拍人像太广）
- 无法实现真正的光学虚化

**结论**：预算 ¥10000+，直接全画幅。预算 ¥5000-10000，APS-C 完全够用。预算有限或不想投入太多学习成本，旗舰手机作为入门完全可以。

### 人像镜头推荐

**50mm — 最通用的人像焦段**

50mm 被称为"标准人像镜头"，视角接近人眼的自然感受。它灵活、轻便、价格亲民，是最值得拥有的第一支人像镜头。

- 全身人像：需要较大退让空间，环境人像利器
- 半身人像（膝盖以上）：f/1.8-f/2.8，全身到半身的最佳焦段
- 面部特写：可用但需要靠近，可能有轻微透视变形

经典选择：
- 50mm f/1.8（各品牌"小痰盂"系列）—— 500-800 元，性价比之王
- 50mm f/1.4（索尼 Planar T* 50mm f/1.4 ZA、适马 50mm f/1.4 DG DN Art）—— 2000-4000 元
- 50mm f/1.2（索尼 GM 50mm f/1.2、佳能 RF 50mm f/1.2 L）—— 10000+ 元，顶级画质

> 拍半身人像，50mm f/1.8 的锐度在 f/2.8 时已经非常出色。不要迷信最大光圈，收一档往往是最佳选择 [8]。

**85mm — 肖像之王**

85mm 是专业人像摄影师的标志性焦段。它的视角窄、压缩感强，能把被摄者从背景中"剥离"出来，同时保持自然的面部比例。

- 半身人像（胸部以上）：f/1.4-f/2.0，经典的头像/半身构图
- 面部特写：f/1.4-f/2.8，极度虚化，背景化为色块

经典选择：
- 85mm f/1.8（各品牌入门款）—— 2000-4000 元
- 85mm f/1.4（索尼 85mm f/1.4 GM、适马 85mm f/1.4 DG DN Art）—— 5000-8000 元
- 85mm f/1.2（佳能 RF 85mm f/1.2 L、尼康 Z 85mm f/1.2 S）—— 15000-20000 元

**注意**：85mm 需要较大的工作空间。室内使用至少需要 3-4 米的距离才能拍半身，小户型慎入 [9]。

**35mm — 环境人像首选**

35mm 是一个被低估的人像焦段。它比 50mm 更广，能把更多环境信息纳入画面，适合讲述人与环境的关系。

- 全身+环境人像：f/2.0-f/4.0，最经典的用法
- 半身人像：需要注意距离，太近会有明显的透视变形（鼻子放大）

35mm 适合：咖啡馆人像、街拍人像、旅行人像、情侣照（需要近距离互动的场景）

**70-200mm f/2.8 — 活动人像与户外利器**

70-200mm f/2.8 被称为"大三元"之一，是人像摄影师的万能工具。

- 70mm 端：接近半身人像视角，f/2.8 提供适度虚化
- 135mm 端：接近 85mm 的肖像效果，空间更灵活
- 200mm 端：极致压缩感，背景虚化最大化，适合户外自然环境人像

它的核心优势不是光圈，而是**变焦的灵活性 + 恒定 f/2.8 + 出色的光学素质**。拍活动、婚礼、户外人像，70-200mm 是唯一不需要换镜头就能完成所有焦段任务的镜头 [10]。

### 光圈与虚化

虚化（Bokeh）不是目的，而是手段。很多初学者陷入"光圈全开崇拜症"，什么都要 f/1.4，结果是一只眼睛实另一只虚、鼻尖实耳朵虚——这是灾难。

**实用光圈建议**：

| 拍摄场景 | 推荐光圈 | 说明 |
|---------|---------|------|
| 全身人像 | f/2.8-f/4.0 | 需要全身清晰，过大光圈会导致身体部分模糊 |
| 半身人像 | f/1.8-f/2.8 | 经典范围，兼顾虚化与清晰 |
| 面部特写 | f/1.4-f/2.0 | 最大虚化，但需精确对焦 |
| 双人合照 | f/2.8-f/4.0 | 两人需同时在景深范围内 |
| 群体合照 | f/5.6-f/8.0 | 确保所有人清晰 |

**影响虚化的四个变量**（不是只有光圈）：
1. **光圈大小**：f 值越小，虚化越强
2. **焦段长短**：焦距越长，虚化越强。200mm f/4 的虚化 > 50mm f/2 的虚化
3. **拍摄距离**：离被摄者越近，虚化越强
4. **背景距离**：背景离被摄者越远，虚化越明显

如果你想最大化虚化效果，同时保持两只眼睛都清晰，最优策略是：使用较长焦段（135-200mm）+ 适度光圈（f/2.8）+ 适当的拍摄距离 [11]。

## 1.3 曝光三要素

### 光圈、快门、ISO 的关系

曝光三要素是摄影的物理基础。不理解它们，你就永远在用"自动挡"——相机替你做决定，而它的决定未必符合你的创作意图。

**光圈（Aperture）**

光圈是镜头内部可调节的孔径大小。用 f 值表示，f 值越小，光圈越大。这是摄影中最反直觉的概念之一——f/1.4 比 f/8 的"孔"更大。

- f/1.4：最大光圈之一，进光量巨大，景深极浅
- f/2.8：大光圈，专业镜头的标志性光圈值
- f/5.6：中等光圈，大多数镜头锐度最佳的区间
- f/11：小光圈，风光摄影常用
- f/16-22：极小光圈，需要三脚架配合

每一档光圈变化，进光量翻倍或减半。f/1.4 → f/2 → f/2.8 → f/4 → f/5.6 → f/8 → f/11 → f/16，每步约一档。

**快门速度（Shutter Speed）**

快门速度是传感器暴露于光线的时间长度。

- 1/8000s：极快，冻结高速运动
- 1/1000s：快，冻结人走/跑步
- 1/250s：中等，手持拍摄的常用速度
- 1/60s：慢速，人像手持的下限
- 1/15s-1s：很慢，需要防抖或三脚架

人像摄影的关键快门速度：
- 室内静态人像：1/60s - 1/125s（配合镜头防抖可以更低）
- 室外自然光人像：1/200s - 1/500s
- 活动人像/儿童：1/500s - 1/1000s（防止运动模糊）
- 闪光灯同步：通常不超过 1/200s-1/250s（帘幕同步模式除外）

**安全快门速度法则**：手持拍摄时，快门速度不应低于焦距的倒数。例如用 85mm 镜头，快门速度至少 1/85s，建议 1/100s 以上。这条法则是针对全画幅的，APS-C 需要再乘以裁切系数 [12]。

**ISO（感光度）**

ISO 是传感器对光线的敏感程度。数值越高，画面越亮，但噪点也越多。

- ISO 100：最低原生 ISO，画质最佳
- ISO 400-800：室内可用范围，现代全画幅几乎无可见噪点
- ISO 1600-3200：暗光环境，全画幅可接受，APS-C 开始明显
- ISO 6400+：极端暗光，需要后期降噪

现代相机的高 ISO 表现已经非常出色。索尼 A7IV 在 ISO 3200 时的画质，比 10 年前的相机 ISO 800 还好 [13]。不要害怕提高 ISO——一张有噪点但清晰曝光正确的照片，远比一张"干净"但曝光不足的照片好。

**三者的平衡关系**

这三者构成了一个三角平衡：改变任何一个，必须至少调整另一个来维持曝光。

- 增大光圈（f 值变小）→ 更多光线 → 可以降低 ISO 或提高快门速度
- 提高快门速度 → 更少光线 → 需要增大光圈或提高 ISO
- 提高 ISO → 更亮画面 → 可以使用更小光圈或更高快门速度

人像摄影中的优先级通常是：**光圈优先（控制虚化）→ 快门速度（防止模糊）→ ISO（补光）**。

### 人像摄影中的曝光策略

**自然光条件下的策略**

晴天直射光（最恶劣条件）：
- 避免正午 11:00-14:00 拍摄
- 让被摄者站在阴影中（建筑阴影、树荫）
- 使用反光板给面部补光
- ISO 100，快门 1/200s-1/500s，光圈 f/2.8-f/5.6

阴天（最佳自然光条件）：
- 阴天的云层相当于一个巨大的柔光箱
- 光线均匀柔和，几乎不需要补光
- ISO 100-200，快门 1/125s-1/250s，光圈 f/2-f/4
- 注意：阴天色温偏高（偏蓝），需要调整白平衡

黄金时段（日出后/日落前 1 小时）：
- 光线温暖、角度低、方向性强
- 最佳人像拍摄时段，几乎没有例外
- ISO 100-400，快门 1/125s-1/250s，光圈 f/2-f/4
- 光线变化快，需要快速调整曝光

**室内人工光策略**

影棚闪光灯：
- ISO 100（固定，最小化噪点）
- 快门 1/125s-1/200s（闪光同步速度）
- 光圈控制亮度 + 虚化

环境光 + 反光板：
- 根据现场光线调整 ISO（通常 400-1600）
- 快门不低于安全速度
- 光圈根据需要设定

**曝光补偿策略**

相机的测光系统有时会"犯蠢"。在逆光条件下拍摄人像时，相机会认为"太亮了"，自动降低曝光，导致人脸变暗。这时需要使用曝光补偿（+0.5 到 +2 EV）来提亮面部。

人像摄影中，"宁过勿欠"原则不一定适用——高光过曝比暗部欠曝更难恢复。RAW 格式下，暗部可以安全提亮 2-3 档，但高光一旦"死白"（溢出），数据就彻底丢失了 [14]。

**实用曝光口诀**：
- 优先控制光圈（决定虚化和风格）
- 快门速度保底线（不糊）
- ISO 最后补（现代相机不用太怕噪点）
- 不确定时，向右曝光（ETTR）但不过曝高光 [15]

## 1.4 对焦技巧

### 眼睛对焦

人像摄影中，对焦只有一个正确的位置：**离镜头最近的眼睛**。

这不是建议，是铁律。眼睛是人类视觉沟通的核心器官——观看者会本能地与照片中人物的眼睛建立"对视"。如果眼睛清晰锐利，整张照片就有了"锚点"，即使其他部分略有模糊，观看者的感知依然是"这张照片很清晰"。反之，如果眼睛模糊，即使其他一切都完美，观看者的第一反应也是"这张照片没对上焦"。

**如何实现精准的眼睛对焦**：

**手动选择焦点法**（适用于所有相机）：
1. 将对焦模式设为"单点对焦"或"区域对焦"
2. 手动移动焦点到被摄者眼睛位置
3. 半按快门对焦 → 构图微调 → 完全按下快门
4. 注意：构图微调时不要前后移动，否则焦点会偏移

**眼控对焦法**（现代旗舰相机）：
索尼 A7R V、佳能 R5 Mark II、尼康 Z8 等旗舰机身已经配备了基于 AI 的人眼/动物眼自动识别对焦。这类系统的准确率极高，在多数情况下可以完全信任 [16]。

设置建议：
- 对焦模式：AF-C（连续自动对焦）
- 对焦区域：广域/自动
- 被摄体识别：人眼优先
- 这样设置后，相机会在你构图的同时持续追踪眼睛，你只需要按快门

**需要注意的陷阱**：

当被摄者侧脸或回头时，两只眼睛距离镜头远近不同。这时候必须对焦在**更近的那只眼睛上**。很多初学者会习惯性地对焦在"更正的那只眼睛"或"更完整的那只眼睛"上，这是错误的。

当使用大光圈（f/1.4-f/1.8）拍面部特写时，景深可能只有几厘米。这意味着从一只眼睛到鼻尖的距离都可能导致失焦。在这种情况下，精确对焦到近眼是唯一正确的选择 [17]。

**对焦后重新构图的问题**：

当你用中心对焦点对焦到眼睛上，然后移动相机重新构图时，如果移动过程中相机有前后位移（这几乎不可避免），焦点就会偏移。在 f/1.4 这种极浅景深下，前后偏移 1-2cm 就可能导致失焦。

解决方案：
- 使用更多对焦点（不是只用中心点），减少重新构图的幅度
- 使用机身的触摸屏拖动对焦（大部分现代相机支持）
- 开启眼控自动对焦，完全避免这个问题

### 追焦与陷阱对焦

**追焦（Focus Tracking / AF-C）**

追焦适用于被摄者在移动的场景：走路、奔跑、转身、与摄影师互动。

设置步骤：
1. 对焦模式设为 AF-C（连续自动对焦）
2. 对焦区域设为"广域"或"追踪"
3. 被摄体识别开启"人"
4. 连拍模式开启（高速连拍）
5. 对焦后持续半按快门或使用 AF-ON 按钮锁定追踪

追焦的成功率取决于：
- 镜头的对焦马达速度（超声波 USM / 线性马达 LDM）
- 对焦算法的预测能力
- 被摄者的移动速度和轨迹是否可预测

推荐追焦表现优秀的镜头：索尼 70-200mm f/2.8 GM II、佳能 RF 70-200mm f/2.8 L IS、尼康 Z 70-200mm f/2.8 VR S [18]。

**陷阱对焦（Trap Focus / Prefocus）**

陷阱对焦是一种"预判性"对焦技术。当你知道被摄者会经过某个位置时，可以提前对该位置对焦，然后等待被摄者进入景深范围时按下快门。

使用场景：
- 被摄者走过特定的门/窗/路口
- 舞台表演者会经过预定的位置
- 运动中的人会经过你预判的"最佳构图位置"

操作方法：
1. 提前对焦到被摄者将要经过的位置（可以对焦到同等距离的物体上）
2. 切换到手动对焦（MF）锁定焦点（或使用"对焦锁定"功能）
3. 等待被摄者进入画面
4. 被摄者到达预定位置时，连拍数张

关键参数：光圈不能太大（f/2.8-f/4.0），确保有一定的景深容错空间。f/1.4 的景深太浅，被摄者稍微偏离预定位置就会失焦 [19]。

**对焦辅助技巧**：

**AF-ON 后键对焦**：将对焦功能从快门按钮分离到 AF-ON 按钮上。这样你可以独立控制"对焦"和"快门释放"两个动作，避免重新构图时的焦点偏移问题。这是专业人像摄影师的标准配置。

**手动对焦峰值辅助**：开启峰值显示功能（现代相机大多支持），在手动对焦时，合焦区域会以高亮颜色标出。这在暗光或大光圈场景中非常有用。

## 1.5 RAW vs JPEG — 为什么人像摄影必须用 RAW

### RAW 格式的本质

RAW 不是一种图片格式，而是传感器采集的原始数据文件。它包含了传感器上每个像素点捕获的所有信息——12-bit 或 14-bit 的色彩深度，完整的动态范围。

JPEG 是相机在 RAW 数据基础上，经过白平衡调整、色彩映射、降噪处理、锐化处理、压缩编码后输出的最终图片。JPEG 是 8-bit 色彩，有损压缩，信息已经被大量丢弃且不可恢复。

打个比方：RAW 是生肉，JPEG 是熟菜。你可以用生肉做出无数种菜，但熟菜只能被加热一次，再加热就糊了。

### 为什么人像摄影必须用 RAW

**第一，后期调色的自由度。** 人像摄影后期流程中最重要的一步是肤色调整。不同的被摄者、不同的光线条件、不同的创作风格，都需要精确控制肤色。RAW 的 12-14 bit 色彩深度意味着每个颜色通道有 4096-16384 个色阶，而 JPEG 的 8 bit 只有 256 个。当你对肤色进行提亮、压暗、色调偏移时，RAW 文件能承受更大的调整幅度而不出现色阶断裂（banding）[20]。

**第二，白平衡可以无损调整。** 拍摄时设定的白平衡在 RAW 中只是一个"标签"——你可以后期随意改变，没有任何画质损失。JPEG 中的白平衡是"烙印"在像素数据中的，大幅度调整会引入色偏和噪点。人像摄影中，白平衡直接影响肤色表现，而准确的白平衡在混合光源（日光 + 钨丝灯 + 荧光灯）环境下几乎不可能在拍摄时就设定正确 [21]。

**第三，高光和阴影的恢复能力。** RAW 文件通常有 2-3 档（甚至更多）的高光和阴影恢复空间。这意味着：
- 过曝的脸部细节可以拉回来（只要不是完全"死白"）
- 欠曝的暗部细节可以安全提亮
- 逆光人像中，天空的高光和面部的暗部可以分别调整

JPEG 的动态范围极其有限，一旦过曝或欠曝超过 0.5-1 档，数据就永久丢失了。

**第四，降噪的空间更大。** 高 ISO 拍摄时，RAW 格式配合专业降噪软件（如 Adobe Lightroom 的 AI 降噪、DxO PureRAW、Topaz DeNoise AI）可以去除大量噪点同时保留细节。JPEG 因为信息已经被压缩丢弃，降噪后的画面容易出现"涂抹感"——细节丢失的同时噪点也未必完全清除 [22]。

**第五，非破坏性编辑。** RAW 的所有后期调整都是"参数化"的——你随时可以回到原始数据重新调整，永远不会"改坏"原图。JPEG 的每次保存都是有损的，编辑 → 保存 → 再编辑 → 再保存，画质会逐次劣化。

### 常见反对意见及其回应

**"RAW 文件太大了，占用存储空间。"**
一张 4500 万像素的 RAW 约 60-80MB，同场景的 JPEG 约 15-20MB。存储空间的价格已经极低——2TB SSD 约 ¥700，能存 25000+ 张 RAW。这不是一个值得妥协的理由。

**"我不会后期。"**
学。人像摄影后期是必选项，不是可选项。Lightroom 的基本面板（曝光、对比度、白平衡、HSL）花一周末就能入门。拒绝学后期的人，等于把创作的一半控制权交给了相机厂商的预设算法。

**"RAW 拍摄速度慢，连拍缓存不够。"**
如果你确实需要 JPEG 的速度（体育摄影、新闻摄影），很多相机支持 RAW+JPEG 同时拍摄。但人像摄影通常不需要高速连拍，RAW 的速度完全够用 [23]。

### 实用设置建议

- 文件格式：RAW（14-bit 如果相机支持）
- 如果担心存储，可以选择压缩 RAW（Lossless Compressed RAW），体积减少 30-40% 且无画质损失
- 不要使用"小 RAW"（S-RAW / M-RAW），这些格式实际上降低了分辨率
- JPEG 只在以下情况考虑使用：(1) 客户需要即时出片 (2) 存储卡/硬盘确实不够 (3) 你完全不打算后期

## 1.6 色彩空间：sRGB vs Adobe RGB

### 什么是色彩空间

色彩空间（Color Space / Color Gamut）定义了一组设备或文件能够表示的颜色范围。把它想象成一个"色域球"——球越大，能包含的颜色越多。

**sRGB**：1996 年由 Microsoft 和 HP 联合制定，是互联网和大多数显示设备的标准色彩空间。它的色域覆盖了大约 35% 的 CIE 1931 色域——换句话说，人眼能看到的颜色中，sRGB 只能表示其中约 35%。

**Adobe RGB (1998)**：Adobe 公司制定，色域比 sRGB 大约 35%，特别是在青绿色（Cyan-Green）区域有明显扩展。它覆盖了大约 50% 的 CIE 1931 色域。

### 人像摄影应该用哪个？

**简短答案：99% 的情况下，用 sRGB。**

**详细解释**：

Adobe RGB 色域更大的优势在理论上是成立的——更多颜色等于更多的表达可能性。但现实中有几个关键问题：

**1. 显示设备的限制**

绝大多数显示器、手机屏幕、平板电脑都是 sRGB 色域。如果你用 Adobe RGB 拍摄并在普通显示器上查看，颜色反而会变得"淡"和"灰"——因为显示器无法显示那些 sRGB 之外的颜色，但文件告诉它"这个颜色应该是更饱和的"，结果就是显示器用了一种更灰的近似色来显示。

只有专业广色域显示器（如 BenQ SW 系列、EIZO ColorEdge 系列、Apple Pro Display XDR）才能正确显示 Adobe RGB 的完整色域。这些显示器价格在 ¥5000-30000+ [24]。

**2. 输出渠道的限制**

如果你的人像照片主要用于以下用途，全部是 sRGB 色域：
- 社交媒体（微信朋友圈、小红书、微博、Instagram）
- 网页展示
- 手机/平板观看
- 大多数在线打印服务

只有在以下场景中，Adobe RGB 才有实际意义：
- 印刷出版（杂志、画册）
- 专业打印（使用支持 Adobe RGB 的打印机和纸张）
- 在专业广色域显示器上展示

**3. 色彩管理的复杂性**

使用 Adobe RGB 意味着你必须做好完整的色彩管理工作链：
- 显示器需要硬件校色（校色仪如 X-Rite i1 Display）
- 编辑软件需要正确识别文件色彩空间
- 输出时需要正确转换为 sRGB（如果目标是网络分享）
- 任何一个环节出错，颜色就会偏

对于大多数人像摄影师来说，这个管理成本是不值得的 [25]。

### 推荐设置

**拍摄时**：sRGB。理由是所见即所得，减少后期困惑。

**如果你确定需要印刷输出**：
- 拍摄时选择 Adobe RGB
- 后期全程在广色域显示器上工作
- 最终输出时再转换为 sRGB（用于网络）或保持 Adobe RGB（用于印刷）

**Lightroom 工作流**：
- Lightroom 内部使用 ProPhoto RGB（比 Adobe RGB 更大的色域），这是 RAW 文件的默认工作空间
- 导出时根据目标选择：网络分享 → sRGB；印刷 → Adobe RGB 或 CMYK
- 你不需要在相机上纠结色彩空间的问题，Lightroom 会自动处理

**关键原则**：色彩空间的选择取决于**输出渠道**，而不是**拍摄设备**。先想清楚你的照片最终在哪里被观看，然后倒推设置。对于 99% 的现代人像摄影作品，答案是 sRGB [26]。

# 第二章：构图的艺术

构图是人像摄影中最具创造性的维度。技术参数可以学习，但构图的审美需要训练和积累。好消息是，构图有规则可循——先学规则，再突破规则。

## 2.1 经典构图法则

### 三分法则及其突破

三分法则是构图的基础中的基础。将画面用两条水平线和两条垂直线均分为九个区域（九宫格），将主体放在线条交叉点上。

**人像摄影中的三分法应用**：

![三分法则人像构图示例](images/composition/01.jpg)

- 眼睛放在上三分线上（最常见、最稳妥的位置）
- 主体身体偏向一侧的垂直三分线，另一侧留白
- 地平线（如果有）放在上三分线或下三分线上

三分法则的本质是**不对称平衡**——主体偏离中心产生张力，留白提供呼吸感。观看者的视线会自然地被引导到交叉点上，这是人类视觉感知的统计学规律（基于眼球追踪研究）[27]。

**突破三分法则**：

三分法则是安全牌，但不是唯一选择。过度依赖它会让所有照片看起来"教科书味"太重。

**中心构图**：将被摄者放在画面正中央。这在以下场景中效果出色：
- 对称环境（走廊、门洞、建筑立面）
- 面部特写（眼神直视镜头时，中心构图产生最强的对视冲击力）
- 低角度仰拍（人物在画面中央，两侧天空对称）

**极端偏移**：将被摄者推向画面的极端边缘，留出大面积的空间。这产生强烈的叙事感——"这个人在看着什么？"或"这个人正要离开"。

**黄金比例构图**：类似于三分法，但分割线按照 1:1.618 的比例放置（而非 1:1:1）。这比三分法更接近自然的视觉和谐比例，但差异微妙，实践中很多人分辨不出两者的区别 [28]。

### 引导线构图

引导线是画面中将观看者视线引向主体的线条元素。这些线条可以是真实的线条（道路、栏杆、走廊、桥梁），也可以是暗示性的线条（光线投射、阴影、颜色渐变）。

![引导线构图示例](images/composition/02.jpg)

**人像摄影中的引导线技巧**：

- **汇聚线**：从画面边缘向主体汇聚的线条（如走廊、道路），产生强烈的纵深感，将观看者的注意力"吸"向被摄者
- **对角引导线**：从画面一角延伸到被摄者的线条，增加动态感
- **弧形引导线**：河流弯曲、道路转弯等弧线元素，比直线引导更柔和自然
- **视线引导**：被摄者看向画面中的某个方向时，观看者会跟随这个"视线线"去看

关键原则：引导线的终点应该是被摄者的眼睛或面部。如果引导线指向了被摄者的脚或者画面外的空白区域，构图就失败了 [29]。

### 框架构图

框架构图是利用画面中自然存在的"框"来包围被摄者。这些框可以是门洞、窗户、拱门、树枝、桥洞、甚至两个人的肩膀。

![框架构图示例](images/composition/03.jpg)

**框架构图的心理效应**：

- **聚焦**：框架像画框一样，将被摄者从复杂的环境中"切割"出来，让观看者的注意力自然集中在框内
- **层次感**：前景的框架 + 中景的主体 + 远景的背景，创造三维空间感
- **窥视感**：框架给观看者一种"偷窥"的感觉，增加照片的故事性和私密感
- **重复**：多个框架（如连排窗户）可以产生节奏感和重复美感

**框架构图的技术要点**：
- 框架本身不需要完全清晰——有时模糊的前景框架反而增加层次感
- 框架的形状和颜色应与主体形成对比
- 避免框架遮挡被摄者的眼睛或面部关键区域
- 框架内的光线应该比框架外的光线更亮，或者主体应该比框架更突出 [30]

### 对称与不对称构图

**对称构图**：

![对称构图示例](images/composition/04.jpg)

对称构图传达稳定、庄严、仪式感。适用于：
- 建筑背景中的人物（站在建筑中轴线上）
- 水面倒影场景
- 正面肖像（面部本身大致对称）

对称构图的风险：容易显得"呆板"。打破对称的方法包括：让被摄者的姿势/表情不完全对称、在光线或色彩上引入微妙的不对称。

**不对称构图**：

![不对称构图示例](images/composition/05.jpg)

不对称构图传达动态、张力、故事性。适用于大多数日常人像场景。

不对称不等于失衡。一张好的不对称照片中，虽然主体偏离了画面中心，但通过以下方式保持视觉平衡：
- **大小平衡**：大的主体靠近中心，小的元素远离中心
- **明暗平衡**：亮的一侧面积小，暗的一侧面积大
- **虚实平衡**：清晰的一侧靠近中心，模糊的一侧远离中心
- **方向平衡**：被摄者面向留白方向，面部的"重量"由留白"承载" [31]

### 对角线构图

![对角线构图示例](images/composition/06.jpg)

对角线构图是最具动态感的构图方式之一。将主体或主要线条沿画面对角线方向排列，产生强烈的运动感和视觉张力。

**人像摄影中的对角线应用**：
- **身体倾斜**：让被摄者的身体呈对角线方向（如斜靠墙壁、侧身站立时身体轴线倾斜）
- **手臂/腿部延伸**：肢体沿对角线方向伸展
- **躺姿/坐姿**：被摄者躺在对角线方向上
- **环境对角线**：楼梯、斜坡、河岸等环境元素沿对角线延伸

对角线构图的关键：方向的"阅读"习惯。从左下到右上的对角线通常感觉"上升"和"积极"，而从左上到右下的对角线感觉"下降"和"消极"。这不是绝对的，但值得在创作时考虑 [32]。

## 2.2 人像摄影专用构图

### 眼睛的位置

眼睛是人像照片的"引力中心"。无论画面中有什么其他元素，观看者的视线总是首先被眼睛吸引。因此，眼睛的位置决定了整张照片的视觉重心。

![眼睛位置构图示例](images/composition/07.jpg)

**经典眼睛位置**：

**上三分之一线**：最常用的眼睛位置。将眼睛放在画面上方三分之一的水平线上，面部和身体占据下方 2/3 的空间。这给被摄者上方留出天空或简洁的背景，同时让面部成为视觉焦点。适用于大多数半身和全身人像。

**黄金分割点**：将眼睛放在左上或右上的黄金分割交叉点上。比三分法更"精确"的视觉和谐位置。

**画面中央**：适用于面部特写和强烈的眼神对视场景。当被摄者直视镜头时，中心放置的眼睛产生最强的"对话感"和"冲击力"。

**需要避免的位置**：
- 画面正中间偏上一点点——不居中也不在三分线上，是最尴尬的位置
- 画面底部——头部被"压"在画面底部，产生压迫感和不适感
- 过度靠近画面顶部——头顶没有留白，感觉被"截断"

**特殊技巧：留出"视线空间"**

当被摄者看向画面中的某个方向时，眼睛前方应该留出比后方更多的空间。如果被摄者向右看，画面右侧的留白应该大于左侧。这条规则适用于任何方向的目光——给视线一个"去处"，观看者才能自然地跟随被摄者的目光方向 [33]。

### 留白与呼吸感

![留白构图示例](images/composition/08.jpg)

留白（Negative Space / White Space）是画面中没有被主体占据的空间。在人像摄影中，留白不是"浪费空间"，而是赋予照片呼吸感、意境和叙事深度。

**留白的功能**：

- **呼吸感**：被摄者周围有足够的空间，照片不显得"挤"。观看者需要视觉上的"休息区"来消化信息。
- **情绪渲染**：大面积留白可以传达孤独、自由、思考、期待等情绪。留白越多，情绪越"安静"。
- **强调主体**：留白就像照片中的"静音"——它不是声音，但它让声音更有力量。
- **叙事空间**：被摄者看向留白方向时，留白成为了"故事将要发生的地方"。

**留白比例的参考**：

| 风格 | 留白占比 | 适用场景 |
|------|---------|---------|
| 紧凑/冲击力 | 10-20% | 面部特写、情绪肖像 |
| 标准 | 30-50% | 大多数人像 |
| 大面积留白 | 60-80% | 艺术人像、情绪表达 |
| 极端留白 | 80%+ | 极简风格、杂志封面 |

**留白的内容同样重要**：
- 留白区域应该是简洁的——纯色、渐变、模糊、均匀纹理
- 留白区域如果过于杂乱，就不再是"留白"而是"干扰"
- 天空是最常见的优质留白区域
- 大面积的墙壁、水面、地面也是不错的选择 [34]

### 裁切位置（关节规则）

![关节裁切规则示意图](images/composition/09.jpg)

裁切位置是初学者最容易犯的错误之一。错误的裁切位置会让照片看起来"被切断"，产生视觉上的不适感。

**关节规则**：永远不要在被摄者的关节处裁切。

具体来说，应该避免的裁切位置：
- 脖子处（把头"切"下来）
- 手腕处（把手"切"下来）
- 肘关节处
- 膝盖处
- 脚踝处

**正确的裁切位置**：

| 构图类型 | 裁切位置 |
|---------|---------|
| 全身照 | 包含头顶到脚底，上方和下方各留少量空间 |
| 七分身（大腿） | 在大腿中部裁切，远离膝盖 |
| 五分身（腰部） | 在腰部裁切 |
| 三分身（胸部） | 在胸部中间偏上裁切，远离肘关节 |
| 半身照（肩膀） | 包含双肩，上方留少量空间 |
| 面部特写 | 额头和下巴各留适当空间，避免裁到下巴尖 |

**这条规则可以被打破吗？** 可以。时装摄影和时尚杂志经常故意在关节处裁切以产生前卫效果。但前提是你知道自己在做什么。对于大多数实用人像摄影，遵循关节规则是最安全的选择 [35]。

### 低角度仰拍

![低角度仰拍示例](images/composition/10.jpg)

低角度仰拍是让被摄者显得更高大、更有气势的构图方式。镜头在被摄者腰部以下甚至更低的位置，向上仰视拍摄。

**低角度的效果**：
- **高大感**：透视效果让被摄者看起来更高
- **力量感**：仰拍暗示被摄者的"权威"和"主导"地位
- **戏剧性**：不常见的视角产生视觉新鲜感
- **天空背景**：低角度自然纳入更多天空，背景更简洁

**低角度的注意事项**：
- 面部比例可能变形——下巴被拉长，鼻子被放大，额头相对变小。45 度以上的极端仰拍尤其明显
- 需要控制变形程度，适度使用
- 配合广角镜头（35mm 或更广）时变形更严重，配合中长焦（85mm+）时变形较小但需要更大仰角
- 被摄者的表情很重要——严肃的表情 + 仰拍 = 威严感；微笑 + 仰拍 = 好莱坞明星感

### 高角度俯拍

![高角度俯拍示例](images/composition/11.jpg)

高角度俯拍与低角度完全相反——镜头在被摄者头部以上，向下俯视拍摄。

**高角度的效果**：
- **可爱/脆弱感**：被摄者看起来更小巧、更柔和
- **亲近感**：俯拍产生一种"照顾"和"亲近"的视觉暗示
- **情绪化**：俯拍常用于表达孤独、沉思、忧郁等情绪
- **环境展示**：更容易纳入地面和周围环境

**高角度的注意事项**：
- 面部比例变化——头顶被拉长，下巴变窄，整体面部比例失真
- 双下巴风险——任何俯拍角度都会让下巴区域出现皮肤堆积，需要被摄者微微抬头或收下巴
- 45 度以上的极端俯拍像"无人机视角"，需要特定的创作意图
- 俯拍时景深变大（因为相机距离面部更远），背景虚化减弱

**高低角度的交替使用**：
在同一次拍摄中交替使用高低角度，能产出视觉多样性极强的成片组。建议：先用平视角度拍"安全牌"，确认基本构图和曝光后，再尝试低角度和高角度的变体 [36]。

## 2.3 景深控制

### 浅景深 vs 深景深的选择

景深（Depth of Field, DoF）是画面中"清晰"的范围。浅景深意味着只有极窄的区域是清晰的，其余部分被虚化；深景深意味着从前景到背景都在清晰范围内。

![浅景深示例](images/composition/12.jpg)
![深景深示例](images/composition/13.jpg)

**浅景深的适用场景**：

- **突出主体**：当背景杂乱无法避开时，浅景深是最好的"背景杀手"
- **情绪肖像**：浅景深产生梦幻、柔和、私密的氛围
- **面部特写**：让观看者所有的注意力集中在面部
- **光线杂乱的环境**：虚化掉不需要的光斑和杂色

**深景深的适用场景**：

- **环境人像**：需要让人物和环境都清晰，传达人与空间的关系
- **群体照**：多人合照需要保证所有人都在景深范围内
- **叙事性人像**：背景中的细节（道具、场景元素）是故事的一部分
- **街拍人像**：街拍中环境信息很重要，过浅的景深会丢失"街"的元素

**景深决策框架**：

| 判断因素 | 浅景深 | 深景深 |
|---------|--------|--------|
| 背景质量 | 背景杂乱/无关 | 背景美丽/与人物相关 |
| 照片用途 | 突出人物本身 | 讲述人与环境的关系 |
| 情绪目标 | 梦幻、亲密、聚焦 | 纪实、叙事、开阔 |
| 被摄者数量 | 1人 | 多人 |

**实用景深公式（简化版）**：

影响景深的四个因素，按影响大小排序：
1. **拍摄距离**（影响最大）：离被摄者越近，景深越浅
2. **光圈**：光圈越大，景深越浅
3. **焦段**：焦距越长，景深越浅
4. **传感器大小**：传感器越大，在相同等效视角下景深越浅

一个常见的误区是"大光圈 = 最好的人像"。实际上，用 85mm f/1.4 拍面部特写时，景深可能只有 2-3cm——这意味着被摄者轻微的前后晃动就可能导致失焦。在实际拍摄中，f/1.8-f/2.8 通常是更"安全"的选择 [37]。

### 焦段对景深的影响

焦段对景深的影响是深刻且容易被误解的。

**核心原理**：相同构图（即被摄者在画面中占据相同比例）下，更长的焦段会产生更浅的等效景深。

举例说明：
- 用 35mm f/1.4 拍半身人像（需要走近）
- 用 85mm f/1.4 拍半身人像（需要退远）
- 用 200mm f/2.8 拍半身人像（需要退更远）

这三张照片中，被摄者在画面中的大小相同，但背景虚化程度完全不同：200mm > 85mm > 35mm。原因是更长焦段需要退得更远，使得背景在画面中的比例被压缩得更小，虚化效果更强 [38]。

**不同焦段的景深特征**：

| 焦段 | 特征虚化风格 | 最佳使用场景 |
|------|-------------|-------------|
| 35mm | 温和的虚化，背景仍有辨识度 | 环境人像、叙事性照片 |
| 50mm | 适中的虚化，平衡主体与背景 | 通用人像、日常拍摄 |
| 85mm | 明显的虚化，背景化为色块 | 肖像、情绪人像 |
| 135mm | 强烈的虚化，背景极度压缩 | 户外人像、唯美风格 |
| 200mm | 极致虚化，背景几乎完全抽象 | 艺术人像、背景杂乱环境 |

**焦段对虚化"质量"的影响**：

除了景深（清晰的深度范围），虚化的"质量"（Bokeh Quality）也受焦段影响：
- 长焦段的虚化光斑更大、更圆、更"奶油般"
- 短焦段的虚化光斑较小、较锐利、有"旋涡感"（取决于镜头设计）
- 光圈叶片数量影响光斑形状——圆形光圈（9-11片叶片）的光斑比六角形（6片叶片）更美观 [39]

> [38] Roger Cicala, "How Focal Length Affects Background Blur," lensrentals.com/blog, 2023.
> [39] DxOMark, "Lens Bokeh Quality: A Comprehensive Guide," dxomark.com, 2023.

## 2.4 负空间运用

![负空间运用示例](images/composition/14.jpg)

负空间（Negative Space）是画面中没有被主体占据的空白区域。在第一章我们已经讨论过留白的概念，这里更深入地探讨负空间作为主动构图工具的运用。

### 负空间的类型

**物理负空间**：真实存在的空间——天空、墙壁、水面、地面、沙漠、雪地。这些是视觉上"空"的区域，但构成了照片不可分割的一部分。

**色调负空间**：通过色调差异创造的空间感——浅色主体在深色背景上，或深色主体在浅色背景上。即使背景不是真正的"空白"，如果它与主体的色调差异足够大，也会产生负空间效果。

**虚化负空间**：大光圈虚化后的背景。虽然背景中可能包含具体物体，但虚化使其变成抽象的色块和光斑，功能上等同于负空间。

### 负空间的情感表达

负空间的面积直接控制照片的情绪强度：

**10-20% 负空间**：紧凑、直接、冲击力强。适用于运动人像、时尚摄影、强调表情和动作的场景。

**30-50% 负空间**：平衡、自然、舒适。这是最常用的范围，适合大多数人像场景。

**60-80% 负空间**：诗意、孤独、内省。大面积的空白让被摄者看起来"小"而环境看起来"大"，产生人相对于世界的渺小感。非常适合海边、沙漠、山顶等开阔环境的人像。

**80%+ 负空间**：极简主义、艺术化、概念化。主体成为画面中微小的存在，负空间成为"主角"。这种构图需要极高的审美判断力——稍有不慎就会看起来像"构图失误"而非"刻意设计" [40]。

### 负空间与方向的互动

负空间的位置不是随意的。它需要与被摄者的"方向"配合：

- **被摄者朝右看** → 右侧留负空间（更大）
- **被摄者向右走** → 右侧留负空间
- **光线从左侧来** → 可以在右侧留负空间（形成亮-暗对比）
- **被摄者面朝镜头** → 负空间可以均匀分布，或偏向被摄者"重心"的反方向

**常见错误**：
- 被摄者向右看但左侧留白更多 → 视线"撞墙"，产生压迫感
- 被摄者的运动方向前方没有空间 → "要走出画面了"的视觉焦虑
- 负空间区域有强烈的视觉元素（鲜艳颜色、文字、图形）→ 负空间变成了"正空间"，与主体竞争注意力 [41]

## 2.5 环境人像构图

![环境人像示例](images/composition/15.jpg)

环境人像（Environmental Portrait）是将被摄者置于其生活的、工作的或与身份相关的环境中进行拍摄。与环境融合的人像比纯背景虚化的人像具有更强的叙事性和信息量。

### 环境人像的核心原则

**环境服务于人物，不是反过来。**

环境人像不是风光照片里恰好有个人，而是人像照片里恰好有环境。二者的区别在于：环境人像中，环境是辅助叙事的配角，人物始终是主角。

### 构图策略

**第一层：明确主体与环境的比例关系**

根据叙事需求，决定人物和环境各占多少画面：
- 人物 70% / 环境 30%：人物主导，环境暗示身份（如厨师在厨房中）
- 人物 50% / 环境 50%：人物与环境平衡（如艺术家在工作室中）
- 人物 30% / 环境 70%：环境主导，人物融入环境（如矿工在矿井中）

**第二层：环境元素的叙事选择**

环境中包含的元素应该与人物身份/故事相关：
- 音乐家的环境：乐器、音箱、舞台灯光、观众席
- 厨师的环境：灶台、食材、刀工操作台
- 作家的环境：书架、打字机、咖啡杯、窗户光线

选择哪些环境元素纳入画面、排除哪些，是环境人像构图的核心决策。每个纳入画面的元素都应该有存在的理由——它帮助讲述故事。

**第三层：光线与环境的协调**

环境人像的光线处理比纯人像更复杂：
- 利用环境中的自然光源（窗户光、工厂天窗、户外场景的黄金时段）
- 环境光不足时，使用闪光灯补光（离机闪 + 柔光箱是常用配置）
- 确保人物面部的光线质量不低于环境——不能"环境好看但人脸暗" [42]

**第四层：景深与环境的清晰度**

- 深景深（f/5.6-f/11）：环境完全清晰，人物与环境同等重要
- 中等景深（f/2.8-f/5.6）：环境可以辨认但不抢戏，人物略有突出
- 浅景深（f/1.4-f/2.0）：环境极度虚化，退化为色调和氛围

### 经典环境人像拍摄清单

- 提前到达场地，观察光线、角度、背景
- 确定至少 3 个拍摄位置（不同背景/光线/构图）
- 与被摄者沟通，了解他们与环境的关系（这里对他们意味着什么）
- 先拍环境确认构图，再引入人物
- 拍摄全身/半身/特写三种景别的变体
- 注意地面——环境人像中地面经常被忽略，但它是画面底部的重要构成 [43]

## 2.6 创意构图技巧

![创意构图示例](images/composition/16.jpg)

掌握了经典构图法则后，是时候探索更自由的创意构图了。以下技巧不是"规则"，而是启发——它们可能在某张照片中效果完美，在另一张中完全失败。关键在于理解它们的原理，然后根据场景灵活运用。

### 镜面/反射构图

利用镜子、水面、玻璃幕墙、金属表面等反射介质进行构图。反射构图产生的"双重视觉"——直接看到的主体和反射中的主体——具有独特的视觉复杂性和艺术感。

- **水面反射**：在平静的水面旁拍摄，利用倒影创造对称构图
- **镜子反射**：通过镜子拍摄，可以同时展示被摄者的正面和背面
- **玻璃反射**：街头的玻璃幕墙、咖啡馆的窗玻璃，可以叠加被摄者和街景
- **金属反射**：金属球、不锈钢表面、汽车外壳，产生变形的反射效果

**技术要点**：对焦时通常需要对焦在实际的反射面上，而不是反射中的"虚像"。大多数自动对焦系统会自动处理这个问题，但如果对焦失败，切换到手动对焦即可 [44]。

### 前景遮挡构图

在镜头和被摄者之间放置前景元素（树叶、花丛、栏杆、烟雾），创造"透过什么东西看人"的效果。

- 前景元素应该虚化（使用大光圈），不能遮挡面部关键区域
- 前景的颜色和形状应该与画面整体色调协调
- 前景遮挡创造"层次感"和"窥视感"，增加照片的深度
- 常用前景：树叶、花瓣、草丛、烟雾、布料、水花 [45]

### 肢体引导构图

利用被摄者的肢体动作（手臂伸展、手指指向、腿部跨步）引导观看者的视线。

- 手臂指向画面中的某个方向，创造动态感
- 手指轻触面部，引导视线到面部
- 背影 + 展开双臂，创造戏剧性的剪影效果
- 舞蹈/运动中的肢体线条本身就是构图的一部分

### 剪影构图

将被摄者拍成纯黑色剪影，背景保持正常曝光。剪影构图强调的是轮廓、形状和姿态，完全去除面部细节。

![剪影构图示例](images/composition/17.jpg)

**剪影拍摄技巧**：
- 光源在被摄者背后（逆光）
- 对准背景的明亮区域测光，然后手动降低曝光 1-2 档
- 被摄者的轮廓必须清晰可辨——手臂、头发、衣物边缘
- 避免被摄者的身体部分与背景中的其他物体重叠（轮廓模糊）
- 最佳时段：日出日落，太阳刚好在地平线或被摄者背后
- 剪影中人物的姿态比表情更重要——身体的线条和张力决定成败 [46]

### 多重曝光构图

将多张照片叠加在同一画面中，创造超现实的视觉效果。现代相机多数内置多重曝光功能，也可以通过后期合成实现。

- **双重人像**：同一被摄者的两个不同姿态叠加，表达内心冲突或时间流逝
- **人像 + 环境**：人像与自然景观（树、山、天空）叠加，表达人与自然的关系
- **运动轨迹**：多张运动中的人像叠加，表达速度和动态

**关键原则**：多重曝光不是为了炫技，而是为了表达单一曝光无法传达的概念。每一次叠加都应该有明确的创作意图 [47]。

### 画面外的空间暗示

有时候，最有力的构图元素不在画面之内。通过被摄者的目光、手势或运动方向暗示画面外的存在，激发观看者的想象力。

- 被摄者看着画面外的某处 → "他在看什么？"
- 手指向画面外 → "那里有什么？"
- 身体向画面外运动 → "他要去哪里？"
- 面部表情暗示与画面外某人的互动 → "他在对谁笑？"

这种构图的核心是"讲故事"——你不展示全部信息，而是给观看者一个线索，让他们自己补完故事。这种参与感和好奇心是静态照片最有力的武器 [48]。
