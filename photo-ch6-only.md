# 第六章 后期处理完整流程

> 拍摄只是完成了人像摄影的一半工作。后期处理——从RAW文件的影调调整到精修细节——决定了最终作品的质感与情绪表达。本章将系统性地讲解从Lightroom基础调色到Photoshop精修、手机APP快速修图、批量处理及导出分享的完整工作流。每一步都有明确的操作参数和实战案例，目标不是教你"怎么用工具"，而是教你"为什么这样调"。

## 6.1 Lightroom人像调色完整流程

### 导入与基础调整

Lightroom是现代数码摄影后期处理的核心平台，其非破坏性编辑模式让摄影师可以反复调整而不损伤原始文件。导入照片后，建议先进入"修改照片"模块，从**基本面板**开始系统性调整。

![Lightroom基础面板](images/editing/01.jpg)

**曝光三要素的精细调整：**

曝光滑块是全局亮度控制，但不要把它当作唯一的调光工具。人像摄影中，正确的做法是分层控制亮度：

- **曝光（Exposure）**：整体画面亮度。人像照片建议先拉到接近肉眼观感，但保留高光和阴影的层次。通常调整范围在 -0.5 到 +0.5 EV之间。如果原始拍摄曝光基本准确，此步骤可能只需要微调 ±0.3EV。
- **对比度（Contrast）**：控制明暗之间的跨度。人像摄影中过高的对比度会让面部阴影生硬，建议初始值设在 +10 到 +25之间。对比度提升会同时压暗阴影并提亮高光——这和后面单独调整高光/阴影滑块的效果叠加，需要注意。
- **高光（Highlights）**：向左降低可以恢复过曝区域（如额头反光、窗户溢光）。人像中最常见的操作是将高光拉到 -30 到 -80 之间，确保面部高光不过曝。这是LR最强大的恢复工具之一，可以拉到 -100 极限值。
- **阴影（Shadows）**：向右提升可以打开暗部细节（如眼窝、鼻翼阴影）。建议 +20 到 +60。但注意：提亮阴影会同时降低画面反差感，需要用对比度和清晰度补偿。
- **白色色阶（Whites）**：控制画面最亮部分的截止点。向右推可以让亮部更"通透"，但推过头会让高光区域"糊"成一片纯白。人像中建议 +10 到 +30，以让皮肤高光区域有明确的高光点而不溢出。
- **黑色色阶（Blacks）**：控制画面最暗部分的截止点。向左拉可以让暗部更深沉，增加画面的"重量感"。但注意观察直方图左端——如果大量堆积在边缘，意味着有信息被裁切。建议 -10 到 -30。

**实用操作流程：**

第一步：将曝光设为0，按住Alt/Option键拖动白色色阶，直到蓝色警告刚好不出现——这是确定白色色阶上限的最精确方法。第二步：同样按住Alt键拖动黑色色阶，找到暗部的极限。第三步：根据直方图分布，微调高光和阴影，确保信息既没有大量溢出也没有集中在中间形成"灰蒙蒙"的效果。

![直方图参考](images/editing/02.jpg)

### 白平衡校正

白平衡决定了画面的整体色温倾向，是人像摄影中最容易被忽视却最重要的基础参数之一。错误的白平衡会直接导致肤色失真——偏黄显脏、偏蓝显冷、偏绿显病态。

![白平衡调整面板](images/editing/03.jpg)

**色温（Temperature）**：滑块向左偏蓝（冷调），向右偏黄（暖调）。人像摄影中，健康的肤色通常需要微暖的色温。自动白平衡（AWB）在复杂光源环境下经常出错，特别是混合光源（日光+钨丝灯、荧光灯+窗户光）场景下。

**手动校准方法：**
1. 使用白平衡选择器（吸管工具），点击画面中理论上应该是中性灰或纯白的位置——白衬衫、眼白、灰色背景纸。注意不要点击过曝区域，因为过曝的像素失去了色彩信息，无法准确校准。
2. 如果没有可靠的中性参考物，以肤色为基准：亚洲人肤色在正确白平衡下，色温通常在 4800K-5500K 之间；欧美人肤色约在 5500K-6500K。
3. 日系清新风格故意偏高色温（5800K-6500K），制造温暖感；电影色调则经常偏低色温（4200K-4800K），制造冷峻感。

**色调（Tint）**：色温解决黄蓝轴问题后，色调解决绿品红轴问题。向左偏绿，向右偏品红。人像中常见的校正是：在荧光灯或树叶反射环境下向右加品红（+5到+15），消除绿色偏移对肤色的影响。

### 肤色还原（HSL面板）

![HSL面板肤色调整](images/editing/04.jpg)

HSL（色相/饱和度/明度）面板是人像调色中控制肤色的核心工具。人的肤色主要由橙色、红色和黄色三个通道构成，精确控制这三个通道可以实现从自然还原到风格化调色的各种效果。

**目标肤色的RGB参考值（16进制）：**
- 健康白皙肤色：#F5D0B0 到 #E8BE9A
- 自然暖调肤色：#E8B88A 到 #D4A070
- 小麦色/蜜糖色：#C49470 到 #A67C5A

**橙色通道（对肤色影响最大，约占60%）：**
- **色相（Hue）**：左移偏红，右移偏黄。+2到+5可以让肤色偏暖偏甜美；-3到-5制造清冷感。
- **饱和度（Saturation）**：降低 -10 到 -25 可以让肤色更"高级感"、更干净；提高会让肤色更鲜艳但容易显"油腻"。建议初学者从降饱和开始。
- **明度（Luminance）**：+10 到 +30 提亮肤色，是"美白"最自然的方式。注意不要超过 +40，否则肤色会失去血液感。

**红色通道（唇色、面部红晕区域）：**
- 色相微调可以改变唇色倾向：-5偏洋红，+5偏橙红
- 饱和度适度提升 +5 到 +15 可以让嘴唇更红润
- 明度 -5 到 -15 可以加深唇色，制造更有质感的妆容效果

**黄色通道（高光肤色区域）：**
- 明度 +10 到 +20 可以提亮面部高光区域
- 饱和度 -10 到 -20 消除皮肤泛黄感，特别适合处理暗光环境下的肤色发黄问题

**关键技巧——分离色调用于肤色：** 如果HSL面板还不够精确，可以使用"分离色调"面板，在高光和阴影分别添加微妙的色彩。例如：高光加微量暖黄（色相45，饱和度5-8），阴影加微量冷青（色相200，饱和度3-5），可以制造非常有层次感的肤色过渡。

### 清晰度与去朦胧

![清晰度与去朦胧效果对比](images/editing/05.jpg)

**清晰度（Clarity）**：这是一个中频对比度增强工具，主要影响画面中间调区域。在风景摄影中经常大幅提升，但在人像摄影中需要极其谨慎——过高的清晰度会让皮肤纹理被过度强化，毛孔和细纹变得非常明显，甚至出现"油画效果"。

**推荐参数：**
- 男性人像：+10 到 +25（适度强化面部轮廓和皮肤质感）
- 女性人像：-5 到 +10（保持皮肤柔和）
- 儿童人像：-10 到 -20（极致柔和）
- 胶片风格：-20 到 -40（模拟胶片低反差特征）

**去朦胧（Dehaze）**：这个工具最初用于去除雾霭和大气散射，但它的实际效果是全局对比度和饱和度的强力提升。参数调整范围极大，效果也极其显著。

- 去朦胧 +10 到 +20：适合灰蒙蒙天气拍摄的照片，可以快速恢复通透感
- 去朦胧 -10 到 -30：制造低对比度的柔和氛围感，适合日系清新风格
- 警告：去朦胧超过 +30 会严重改变肤色，橙色和红色通道会变得过度饱和。如果需要高去朦胧值，建议回到HSL面板相应降低橙色和红色饱和度作为补偿

### 预设（Preset）制作与使用

![预设管理面板](images/editing/06.jpg)

预设是Lightroom效率提升的核心工具，本质上是一组参数的快照。正确使用预设可以大幅减少重复操作时间，同时保证同一组照片的视觉一致性。

**制作个人预设的黄金法则：**

1. **基础修正预设（Base Correction）**：只包含白平衡校正、镜头矫正、去除色差、启用镜头配置文件。这是所有照片都应该应用的第一步。制作方法：对一张在标准光源下拍摄的灰卡照片进行校正，然后保存为预设。这个预设不包含任何风格化参数。

2. **风格预设（Style Preset）**：包含调色、色调曲线、HSL、分离色调等风格化参数，但不包含曝光和白平衡——因为每张照片的拍摄条件不同，曝光和白平衡必须单独调整。制作方法：先将基础修正预设应用，然后在一组典型照片上调出满意的风格，保存时不勾选"白平衡"和"处理版本"。

3. **预设的命名规范**：`类别_风格_强度.版本`，例如 `Portrait_CleanWarm_Medium.v2`。避免使用"我的最爱"这类无意义的名称。

4. **预设的参数建议**：单个预设中修改的滑块数量不要超过15个。超过这个数量说明预设过于复杂，对不同照片的适应性会急剧下降。最有效的预设通常只改5-8个核心参数。

**使用预设的正确流程：**
1. 应用基础修正预设
2. 单独调整白平衡和曝光（根据每张照片实际情况）
3. 应用风格预设
4. 微调（肤色、局部调整）
5. 保存为"定制预设"用于同组照片

### 具体调色案例

![四种调色风格对比](images/editing/07.jpg)

**案例一：日系清新风格**

日系清新的核心特征是低对比度、偏暖色温、高亮度、低饱和度，整体呈现柔和透亮的空气感。

- 曝光：+0.3 到 +0.8
- 对比度：-15 到 -25
- 高光：-20 到 -40
- 阴影：+40 到 +70
- 色温：+10 到 +20（偏暖）
- 色调：+5 到 +10（微偏品红，消除绿色）
- 清晰度：-10 到 -20
- 去朦胧：-15 到 -25
- HSL：橙色明度 +20、饱和度 -15；红色明度 +10、饱和度 -10
- 色调曲线：高光端下弯（压低高光），阴影端上抬（提亮暗部），形成"反S曲线"
- 颗粒：添加量 15-25、大小 25-40、粗糙度 40-60（模拟胶片颗粒感）

**案例二：复古胶片风格**

复古胶片的核心是模拟经典彩色负片（如Kodak Portra 400或Fuji 400H）的色彩特征：暗部偏青绿、高光偏暖黄、整体饱和度略低于数码标准、有明显的胶片颗粒。

- 曝光：+0.2 到 +0.5
- 对比度：-5 到 -10
- 色温：+5 到 +15
- 清晰度：-15 到 -30
- 去朦胧：-10 到 -20
- HSL：红色色相 +5（偏橙红）、饱和度 -10；橙色饱和度 -15；绿色色相 -10（偏青绿）；蓝色饱和度 -20
- 分离色调：高光加暖（色相50，饱和度8）；阴影加冷（色相210，饱和度6）
- 颗粒：添加量 30-50、大小 30-50、粗糙度 50-70
- 暗角：数量 -20 到 -35（模拟老镜头暗角）

**案例三：韩式甜美风格**

韩式人像的特点是皮肤白皙通透、整体亮度高、对比度低、色彩偏粉嫩、背景虚化柔和梦幻。

- 曝光：+0.5 到 +1.0
- 对比度：-20 到 -30
- 高光：-30 到 -50
- 阴影：+50 到 +80
- 色温：+5 到 +10
- 色调：+10 到 +20（偏品红，制造粉色氛围）
- 清晰度：-15 到 -25
- HSL：橙色明度 +30、饱和度 -20；红色明度 +15、饱和度 +5（唇色略提）；蓝色饱和度 -10（背景去杂色）；紫色明度 +15（增加梦幻感）
- 色调曲线：高光端显著下弯，整体上抬
- 暗角：样式"高光优先"，数量 -15 到 -25
- 色彩效果面板（可选）：在Photoshop中叠加一层低透明度的粉色渐变

**案例四：电影色调**

电影色调的核心是青橙对比（Teal & Orange）——这是好莱坞商业电影中最常见的调色方案，原理是将肤色推向橙色，将互补色（背景、阴影）推向青色/蓝绿色，利用色彩对比让人物从背景中"跳出来"。

- 曝光：-0.1 到 +0.2
- 对比度：+15 到 +30
- 高光：-15 到 -30
- 阴影：+10 到 +30
- 色温：-5 到 -15（偏冷）
- 清晰度：+5 到 +15
- 去朦胧：+10 到 +20
- HSL：橙色饱和度 +15 到 +25、明度 +10；蓝色色相 -10（偏青）、饱和度 +20；绿色色相 -15（偏青绿）、饱和度 +10
- 色调曲线：RGB通道中阴影端微微下压（增加暗部密度），高光端适度下压（防止过曝）；蓝色通道阴影端上抬（暗部加青蓝），红色通道高光端上抬（亮部加暖）
- 分离色调：高光（色相35，饱和度10-15，暖橙）；阴影（色相195，饱和度8-12，冷青）
- 暗角：数量 -30 到 -50，中点 40-50（明显的镜头暗角增加电影感）

## 6.2 Photoshop精修技术

Lightroom负责全局调色和基础修饰，Photoshop负责像素级的精修工作。商业人像摄影中，PS精修通常占总后期时间的60%-80%。本节讲解最核心的精修技术。

### 液化工具（自然塑形，保持比例）

![液化工具界面](images/editing/08.jpg)

液化（Liquify）是PS中最强大的形体塑形工具，但也是最容易"翻车"的工具——过度液化会让人脸比例失衡，产生不自然的"假人感"。关键原则是：**微妙修正为主，大幅变形为辅**。

**操作前的准备工作：**
1. 复制背景图层（Ctrl/Cmd + J），所有液化操作在新图层上进行
2. 在液化滤镜中选择"高级模式"
3. 展开左侧工具栏中的"人脸识别液化"选项——PS CC 2015.5及以后版本支持自动识别人脸特征点，可以精确控制眼睛、鼻子、嘴巴、脸型的各个参数

**人脸识别液化参数建议：**
- **眼睛大小**：100% 到 107%（超过107%开始明显不自然）
- **眼睛高度**：100% 到 105%
- **眼睛宽度**：100% 到 103%
- **鼻子宽度**：100% 到 95%（适当收窄）
- **嘴唇厚度**：上唇 100% 到 110%，下唇 100% 到 108%
- **脸型宽度**：100% 到 97%（两侧各1.5%，约等于3%总宽度减少）
- **额头高度**：100% 到 103%（仅当额头显得过短时）
- **下巴长度**：100% 到 105%（仅当下巴显得过短时）

**手动液化工具的使用：**
- **向前变形工具（Warp）**：最常用的推拉工具。画笔大小应覆盖需要修改区域的1.5-2倍。压力建议 30%-50%，密度 70%-80%。操作时沿肌肉纹理方向推动，不要横向拉扯皮肤。
- **重建工具**：液化过度的局部恢复。使用"重建"选项中的"恢复全部"可以一键回到液化前状态。
- **冻结蒙版工具**：不需要变形的区域涂抹冻结，避免在调整周围区域时误改关键部位。

**重要原则：**
1. 先整体后局部——先调整脸型轮廓，再调整五官
2. 频繁切换前后对比（空格键+取消勾选"显示图像"），保持客观判断
3. 倾斜画面到正常观看角度检查（很多人在100%缩放下液化看不出问题，缩小到实际展示尺寸后比例失调就暴露了）
4. 保存为智能对象（Smart Object）——这样液化操作始终可以修改或撤销

### 频率分离磨皮（详细步骤）

![频率分离磨皮效果对比](images/editing/09.jpg)

频率分离（Frequency Separation）是专业人像修图的标准技术，核心原理是将图像分为两个独立图层：**低频层（Low Frequency）**包含颜色和光影信息，**高频层（High Frequency）**包含纹理和细节信息。分开处理后合并，可以同时实现光滑肤色和保留皮肤纹理——这是传统模糊磨皮无法做到的。

**详细操作步骤（Photoshop CC）：**

**第一步：准备工作**
1. 打开图像，按 Ctrl/Cmd + J 复制两次，得到三个图层
2. 从上到下重命名为：高频层、低频层、原图层
3. 暂时关闭高频层的可见性（点击眼睛图标）

**第二步：创建低频层（颜色和光影）**
1. 选中低频层
2. 选择菜单 滤镜 > 模糊 > 高斯模糊
3. 模糊半径设为 **4-8像素**（4K分辨率图像可以设到 10-15像素）。关键判断标准：皮肤毛孔和细纹刚好完全消失，但面部光影轮廓（明暗过渡）仍然清晰可见
4. 如果模糊过度（光影都糊了），降低半径；如果模糊不足（毛孔还可见），增加半径

**第三步：创建高频层（纹理和细节）**
1. 重新打开高频层的可见性，选中它
2. 选择菜单 图像 > 应用图像（Image > Apply Image）
3. 参数设置：
   - 源：当前文档
   - 图层：低频层
   - 混合模式：**减去（Subtract）**
   - 缩放：2
   - 补偿值：128
4. 点击确定。此时高频层看起来像一张灰色的"浮雕"图像——这是正常的
5. 将高频层的混合模式改为 **线性光（Linear Light）**
6. 此时画面应该看起来和原图完全一样——检查一下，如果一样说明操作正确

**第四步：低频层处理（颜色修正）**
1. 关闭高频层可见性，选中低频层
2. 选择画笔工具（B），设置：硬度 0%（柔边圆笔刷），不透明度 15-25%，流量 100%
3. 按住 Alt/Option 点击低频层，创建一个剪切蒙版图层到低频层上方，命名为"颜色修正"
4. 使用吸管工具吸取目标肤色，然后用画笔涂抹需要修正的区域：
   - 红色痘印、泛红区域：用正常肤色覆盖
   - 黑眼圈：用周围正常肤色提亮
   - 嘴角暗沉：提亮至与周围一致
   - 肤色不均匀：统一涂抹至一致色调
5. 注意笔刷大小要足够大，操作要轻柔——低频层的目的是让颜色过渡自然，不是粗暴覆盖

**第五步：高频层处理（纹理修饰）**
1. 打开高频层可见性，选中它
2. 同样创建一个空白图层，剪切蒙版到高频层上方，命名为"纹理修饰"
3. 选择仿制图章工具（S），设置：
   - 源："当前和下方图层"
   - 画笔：柔边圆笔刷，硬度 0%
   - 不透明度：15-30%
   - 样本：所有图层
4. 修除明显的皮肤瑕疵：痘痘、雀斑（如果客户要求去除）、明显疤痕
5. **关键：不要去除所有毛孔和细纹！** 保留皮肤的自然纹理。判断标准是：在50%缩放比例下看不到明显瑕疵即可，不要放大到200%去"找茬"
6. 对于较大的瑕疵，先降低不透明度到 10-15%，多次轻柔覆盖

**第六步：最终合并与检查**
1. 在两个剪切蒙版图层上方新建一个合并观察图层（Shift+Ctrl/Cmd+Alt+E）
2. 放大到100%检查：纹理是否保留完整？颜色过渡是否自然？有无明显的修图痕迹？
3. 缩小到实际展示尺寸再检查一次

**频率分离的高级变体：**
- **D&B（Dodge & Burn）配合频率分离**：在低频层上方用曲线调整图层+蒙版进行光影雕刻，可以塑造面部立体感
- **三次频率分离**：将低频层再次分离，得到超低频（整体光影）、中频（过渡）、高频（细节）三层，适合需要极端光影控制的商业级精修

### 眼睛精修

![眼睛精修前后对比](images/editing/10.jpg)

眼睛是人像摄影的焦点区域，精修眼睛可以显著提升整张照片的"神采"。但同样需要克制——过度精修会让眼睛看起来像玻璃珠一样假。

**提亮眼白：**
1. 创建曲线调整图层，拖拽曲线使整体变亮
2. 用黑色画笔涂抹蒙版，只露出眼白区域
3. 适当降低不透明度到 40-60%
4. 注意不要让眼白变得过亮过白——真实的眼白带有微妙的蓝色/红色血丝，完全纯白反而失真

**加深虹膜（瞳孔颜色）：**
1. 创建色相/饱和度调整图层
2. 选择"着色"模式，选择接近瞳孔颜色的色相
3. 用黑色蒙版遮住全部，然后用低不透明度画笔只露出虹膜区域
4. 不透明度 20-40%，让瞳孔颜色更深邃
5. 或者使用"加深工具"（O），范围设为"中间调"，曝光度 5-10%，在虹膜边缘轻轻涂抹增加层次感

**添加眼神光（Catchlight）：**
1. 新建空白图层
2. 选择画笔工具，设置：白色，柔边圆笔刷，不透明度 30-50%，大小约 5-15px
3. 在瞳孔中已有的高光点附近点一个白色圆点
4. 眼神光的位置应该模拟光源方向——如果在左上方有窗光，眼神光应该在瞳孔的左上方
5. 混合模式改为"滤色"，调整不透明度直到自然
6. 注意：每只眼睛只需 1-2 个眼神光点，太多会显得眼睛像玻璃珠

**睫毛处理：**
- 如果原片睫毛清晰：使用加深工具适度加粗睫毛根部，使用液化工具的"向前变形"小笔刷梳理杂乱的睫毛方向
- 如果原片睫毛稀疏：使用极细画笔（1-2px），不透明度 30-50%，在睫毛生长方向手动添加，注意从根部到尖端逐渐变细
- 检查左右眼睫毛密度和方向是否对称

### 牙齿美白

牙齿美白是人像精修中需求很高的项目，但最容易犯的错误是把牙齿处理成不自然的"荧光白"。

**正确的牙齿美白流程：**
1. 使用套索工具（L）或钢笔工具（P）精确选中牙齿区域——注意要将牙龈和嘴唇排除在外
2. 按住 Shift 选择多颗牙齿时保持边缘贴合
3. 创建色相/饱和度调整图层（会自动添加基于选区的蒙版）
4. 选择"黄色"通道，降低饱和度 -20 到 -40
5. 选择"红色"通道，降低饱和度 -5 到 -10（去除牙龈泛红的影响）
6. 创建曲线调整图层，轻微提亮（不要超过 +15 的幅度）
7. 蒙版羽化 1-2px，确保边缘过渡自然

**注意事项：**
- 牙齿不是纯白的，健康的牙齿呈淡黄色到灰白色。美白的目标是去除明显的黄色/咖啡渍，不是变成白纸
- 嘴巴内部（口腔暗部）不要一并提亮——那是应该保持暗调的区域
- 每次调整后放大到 100% 检查牙齿表面纹理是否保留

### 头发处理（飞发修复、发色调整）

**飞发修复：**

飞发是指那些不受控制地飘在面部或背景上的细碎头发。少量飞发增加自然感，过多则显杂乱。

1. **判断是否需要处理**：少量（3-5根）飘在空旷背景上的飞发可以保留，它们增加动态感。横穿面部的飞发、大面积散乱的飞发需要处理。
2. **仿制图章修复**：选择仿制图章工具（S），设置"对所有图层取样"，柔边笔刷，不透明度 80-100%，按住 Alt 在飞发附近的背景取样后覆盖
3. **修复画笔**：适合处理背景均匀区域的细小飞发
4. **Content-Aware Fill（内容识别填充）**：适合较大面积的飞发聚集区域。框选飞发区域后选择"编辑 > 填充 > 内容识别"
5. **手动添加头发**：对于发型缺口，新建图层，用细画笔（1-3px）手动绘制发丝。关键技巧：每根发丝要独立一笔完成，从粗到细，弧度自然。用深浅不同的颜色交替绘制模拟头发层次。

**发色调整：**
1. 选中头发区域（色彩范围选择头发色调，或手动蒙版）
2. 使用色相/饱和度或曲线调整图层
3. 增加发色层次感：创建新图层，混合模式"柔光"，用深色画笔在头发暗部加深，浅色画笔在头发亮部提亮
4. 发色与整体色调统一：如果整体调色偏暖，发色也应随之微调偏暖

### 皮肤纹理保留技巧

![皮肤纹理保留对比](images/editing/11.jpg)

过度光滑的皮肤是业余修图最明显的标志。专业修图追求的是"看起来皮肤很好"，而不是"看起来被修过"。

**保留纹理的核心原则：**

1. **频率分离是最佳方案**：如前文所述，低频层处理颜色、高频层处理纹理——这种方式天然保留了皮肤纹理
2. **磨皮插件的使用**：
   - **Portraiture**（Imagenomic）：最流行的人像磨皮插件，可以精确识别皮肤区域并保留非皮肤区域的纹理。建议"平滑度"设为 10-20，不要超过 30
   - **Retouch4Me**：AI驱动的自动修图插件，磨皮效果自然，适合快速出图
   - **DR4/DR5**（Delicious Retouch）：面板式工具集，包含D&B自动化等实用功能

3. **Dodge & Burn（减淡加深）技法**：
   - 在所有磨皮完成后，使用 50% 灰图层 + 柔光混合模式
   - 用白色画笔（不透明度 3-5%）提亮需要高光的区域：额头、鼻梁、颧骨、下巴
   - 用黑色画笔（不透明度 3-5%）加深阴影区域：发际线、鼻翼两侧、下颌线
   - 这一步的目的是增强面部立体感，不是改变肤色——所以画笔不透明度必须很低

4. **最终纹理叠加**：如果前面的磨皮操作丢失了过多纹理，可以在最终图层上方叠加一层从原图提取的纹理：
   - 复制原图层，放在最上方
   - 混合模式改为"线性光"
   - 不透明度 10-20%
   - 添加黑色蒙版，用低不透明度白色画刷只在皮肤区域露出纹理

## 6.3 手机修图APP推荐与教程

![手机修图APP图标集](images/editing/12.jpg)

不是所有人像照片都需要桌面级精修。对于社交媒体分享和快速出片，手机APP已经足够强大。关键在于选对工具并掌握每个工具的核心功能。

### VSCO（滤镜选择与调整）

VSCO是移动端最经典的胶片模拟滤镜APP，以克制、高质量的滤镜著称。它不是功能最全的修图APP，但在色彩风格模拟方面几乎没有对手。

**滤镜选择策略：**
- **C系列（Color Negative）**：模拟经典彩色负片。C1 偏暖低调，适合室内人像；C7 偏冷灰绿，适合街拍人像
- **A系列（Analog）**：模拟正片/幻灯片。A6 高饱和高反差，适合色彩鲜明的日系人像；A4 低调偏灰，适合情绪片
- **F系列（Instant Film）**：模拟拍立得。F2 偏暖偏粉，适合日系甜美风
- **HB/H系列**：高对比黑白，适合黑白人像
- **V系列**：Vivid vivid，经典VSCO风格——偏绿偏灰，辨识度极高

**调整技巧：**
1. 先选滤镜，再微调参数——不要反过来
2. 滤镜强度不要超过 8（满分12），超过这个值通常显得"滤镜味"太重
3. 核心微调参数：曝光 ±1、对比度 -1 到 +1、饱和度 -1、色温 ±1。VSCO的调整参数非常敏感，每次只调 1-2 个单位
4. 保存自定义预设：调好一张照片后，可以将参数组合保存为自定义预设，用于同组照片保持一致性

### Snapseed（工具使用技巧）

![Snapseed工具面板](images/editing/13.jpg)

Snapseed是Google推出的免费修图APP，功能强大程度远超大多数人想象。它最大的优势是**局部调整能力**——这是很多付费APP都不具备的。

**核心工具详解：**

**选择性调整（Selective）：**
- 点击画面添加控制点，每个控制点只影响周围一定范围内的像素
- 可以精确控制影响范围（双指缩放红色蒙版区域）
- 适合：局部提亮面部、压暗背景、局部调整色温
- 技巧：在人脸上放置一个控制点，提亮 +0.3，饱和度 -0.2，可以让面部瞬间"干净"

**修复工具（Healing）：**
- 类似PS的仿制图章+修复画笔
- 适合去除面部痘痘、飞发、背景杂物
- 用手指涂抹需要去除的区域，松手后自动采样填充
- 注意：涂抹面积不要太大（不超过屏幕宽度的1/4），否则算法容易出错

**双重曝光（Double Exposure）：**
- 可以叠加两张图片，创造艺术效果
- 适合：在剪影人像中叠加风景/花草纹理
- 使用"水彩"或"叠加"混合模式效果最佳

**曲线工具（Curves）：**
- 支持RGB和单独通道调整
- 是移动端最接近桌面级调色精度的工具
- 建议学习基础的S曲线和反S曲线操作

** HDR Scape：**
- 不是HDR合成，而是局部色调映射
- 适合逆光人像——可以同时保留高光细节和阴影层次
- "自然"模式强度 +20 到 +40 即可，"戏剧"模式适合特殊效果

**工作流建议：**
1. Snapseed做结构性调整（曝光、局部提亮、修复瑕疵）
2. VSCO做风格化滤镜
3. 两个APP配合使用效果远好于只用其中一个

### Lightroom Mobile（同步桌面预设）

Lightroom Mobile是唯一可以与桌面版无缝同步的移动修图APP，这是它的核心优势——你在电脑上调好的预设，手机上直接使用。

**预设同步设置：**
1. 确保桌面版和移动版登录同一个Adobe账号
2. 桌面版中：Lightroom Classic > 编辑 > 首选项 > 预设 > Lightroom预设 > "与Lightroom Mobile同步"
3. 手机端：设置 > 生产力 > 同步 > 开启
4. 首次同步可能需要等待几分钟，之后新预设会实时同步

**移动版独有优势：**
- **选择性编辑**：类似桌面版的调整画笔，用手指在屏幕上涂抹需要调整的区域
- **自动蒙版**：可以自动识别人物主体，方便做背景虚化或背景调色而不影响人物
- **RAW文件编辑**：支持直接编辑手机拍摄的RAW格式文件
- **云同步**：手机调好的照片自动出现在桌面版的"同步"收藏夹中

### 醒图（一键美颜vs手动精修）

醒图是字节跳动推出的修图APP，在国内用户量极大。它的核心卖点是AI驱动的美颜功能和丰富的模板。

**一键美颜的使用建议：**
- "智能优化"功能效果不错，但建议降低强度到 40-60%
- 美白不要超过 30%（在"肤色"选项中），否则肤色失去血液感
- 瘦脸不要超过 20%，大眼不要超过 15%
- 磨皮选择"高级磨皮"而非"普通磨皮"——前者保留更多皮肤纹理

**手动精修功能推荐使用：**
- **面部重塑**：醒图的面部重塑比美图秀秀更自然，支持精确到每个面部特征点的调整
- **滤镜+调整**：醒图的滤镜质量整体不错，特别是"日系"和"电影"类别
- **图层功能**：醒图支持多图层编辑和混合模式——这在手机APP中非常少见
- **局部调色**：支持对画面不同区域分别调色，功能接近桌面级

### 美图秀秀使用建议

美图秀秀在中国市场的用户量是所有修图APP中最大的。虽然它的定位偏向"一键美化"和社交分享，但在某些功能上有独到之处。

**美图秀秀适合的场景：**
- 快速社交媒体出图——打开即修，30秒完成
- 人像抠图和换背景功能在手机APP中效果出色
- 拼图、海报模板非常丰富
- 视频修图功能方便短视频内容创作者

**不建议使用的场景：**
- 专业人像精修——美颜算法过于激进
- 需要保留皮肤纹理的场景——磨皮会去除所有毛孔
- 需要精确色彩控制——缺少HSL面板和曲线工具

**使用建议：**
- 如果必须使用美图秀秀，优先使用"专业模式"而非"一键美化"
- 所有美颜参数降低到默认值的 50% 以下
- 导出时选择最高画質选项
- 完成后在其他APP中做最终微调

**各APP对比总结：**

| 功能 | VSCO | Snapseed | LR Mobile | 醒图 | 美图秀秀 |
|------|------|----------|-----------|------|----------|
| 滤镜质量 | ★★★★★ | ★★★ | ★★★★ | ★★★★ | ★★★ |
| 局部调整 | ★★ | ★★★★★ | ★★★★ | ★★★ | ★★ |
| 美颜功能 | ★ | ★★ | ★★★ | ★★★★★ | ★★★★★ |
| 桌面同步 | ★ | ★ | ★★★★★ | ★ | ★ |
| 学习成本 | 低 | 中 | 高 | 低 | 极低 |
| 适合人群 | 摄影师 | 进阶用户 | 专业用户 | 社交媒体 | 快速出图 |

## 6.4 肤色统一技术

![肤色统一前后对比](images/editing/14.jpg)

肤色统一是人像摄影后期中最被低估的技能之一。同一组照片中，由于光线变化、拍摄角度差异、白平衡波动，人物肤色经常不一致——放在一个版面或相册中时，这种不一致会非常明显。

**肤色不一致的常见原因：**

1. **光源色温变化**：从自然光过渡到人造光源，或云层遮蔽阳光的瞬间
2. **环境色反射**：靠近绿色植物会偏绿，靠近红墙会偏红，蓝色天空作为光源则偏蓝
3. **曝光差异**：曝光不足的皮肤显得暗黄，过曝的皮肤显得苍白
4. **后期风格预设**：不同预设对肤色的处理方式不同，直接导致同组照片肤色不统一

**肤色统一的Lightroom方法：**

1. **建立肤色基准**：从同组照片中选出一张白平衡最准确、肤色最自然的照片作为基准
2. **同步白平衡**：选中基准照片，按住Ctrl/Cmd点击其他需要统一的照片，点击"同步设置"，只勾选"白平衡"
3. **使用HSL面板统一**：在基准照片上调整好HSL面板中的橙色、红色、黄色通道参数，然后将这三个参数同步到其他照片
4. **肤色目标值法**：使用Lightroom的"目标调整"工具（HSL面板左上角的圆形图标），在照片中肤色区域上下拖动，直接调整该区域的色相/饱和度/明度

**Photoshop高级肤色统一法：**

1. **色彩采样对比**：在基准照片和需要修正的照片中分别取样肤色（吸管工具），记录RGB值
2. **匹配颜色功能**：图像 > 调整 > 匹配颜色，选择基准照片作为源图像。这个功能可以自动匹配两张照片的整体色彩风格
3. **渐变映射统一**：创建渐变映射调整图层，选择"紫橙"或自定义肤色渐变，混合模式设为"颜色"，不透明度 10-20%。这种方法可以在不改变明暗的前提下统一色调倾向
4. **Lab模式精确统一**：
   - 转换到Lab颜色模式（图像 > 模式 > Lab颜色）
   - 在"a"通道（绿红轴）和"b"通道（蓝黄轴）进行微调
   - Lab模式的优势是将亮度信息（L通道）和色彩信息（a、b通道）完全分离，调色不影响明暗
   - 调整完成后转回RGB模式

**批量肤色的技巧：**

- 为每种常见光源条件预设一组HSL参数（日光、阴天、室内暖光、荧光灯）
- 拍摄时随身携带灰卡，在每组拍摄开始时拍一张灰卡照——后期可以一键校准白平衡，然后同步到同组所有照片
- 使用Lightroom的"参考视图"功能（视图 > 参考视图），将基准照片放在左侧作为参考，右侧实时调色直到肤色一致

## 6.5 色彩分级（Color Grading）

![色彩分级效果展示](images/editing/15.jpg)

色彩分级（Color Grading）是电影工业中的标准术语，指的是为画面不同亮度区域赋予不同的色彩倾向。Lightroom CC 2020引入了专用的"色彩分级"面板（取代了旧的"分离色调"面板），让这一技术变得更容易上手。

**色彩分级与色彩校正的区别：**
- **色彩校正（Color Correction）**：让颜色回归"正确"——白平衡校准、去色偏、还原真实肤色。这是技术层面的工作。
- **色彩分级（Color Grading）**：故意给画面赋予特定色调——为了情绪表达、风格统一、视觉引导。这是艺术层面的工作。
- 工作顺序：先校正，后分级。永远不要在白平衡不准确的情况下做色彩分级。

**Lightroom色彩分级面板详解：**

面板中有三个色轮，分别对应画面的三个亮度区域：
- **阴影（Shadows）**：画面最暗部分的色调
- **中间调（Midtones）**：画面中等亮度区域的色调
- **高光（Highlights）**：画面最亮部分的色调

每个色轮的操作方式：在色轮上点击并拖动，离中心越远色彩越浓。拖动方向决定色相（上方偏黄/红，下方偏蓝/绿，左方偏绿/青，右方偏红/品红）。

**常用色彩分级方案：**

**方案一：青橙对比（Teal & Orange）**
- 阴影色轮：色相 200（青色），饱和度 15-25
- 中间调色轮：不添加或极微弱
- 高光色轮：色相 35（橙色），饱和度 10-20
- 适用：商业人像、户外人像、需要人物突出的场景

**方案二：冷暖对比**
- 阴影色轮：色相 220（蓝色），饱和度 10-15
- 高光色轮：色相 45（暖黄），饱和度 8-12
- 适用：日落人像、城市夜景人像、情绪片

**方案三：单色调（Monochrome Tint）**
- 三个色轮选择同一色系的不同深浅
- 例如：阴影深青（色相 200，饱和度 20），中间调浅青（色相 195，饱和度 8），高光微青（色相 190，饱和度 3）
- 适用：情绪化创作、时尚编辑片

**方案四：暗调复古**
- 阴影色轮：色相 270（紫色），饱和度 8-12
- 高光色轮：色相 30（金色），饱和度 10-15
- 全局饱和度降 10-15%
- 适用：复古风格、文艺写真

**Photoshop色彩分级进阶技法：**

1. **色彩平衡调整图层**：选择"阴影/中间调/高光"分别调整，保留发光度选项勾选（保持亮度不受影响）
2. **渐变映射**：创建渐变映射图层，混合模式"颜色"或"柔光"，不透明度 5-15%。使用从阴影色到高光色的自定义渐变
3. **曲线分通道调色**：在RGB曲线中分别切换到红/绿/蓝通道，在阴影端和高光端分别调整——这是最精细的分级方式
4. **Solid Color + Blending Mode**：新建纯色填充图层，选择目标色调，混合模式"颜色"或"柔光"，不透明度 5-10%。配合蒙版控制影响范围

**色彩分级的核心原则：**
1. **少即是多**：饱和度永远比你想的再低一点。5-15% 的微妙分级比 30% 的明显分级更"高级"
2. **一致性高于一切**：一组照片的色彩分级方案必须统一——即使不同照片的拍摄条件不同，分级风格也要一致
3. **服务叙事**：冷暖色调有情绪暗示——暖色传达安全、亲密、舒适；冷色传达距离、神秘、忧郁。分级方案应该服务于照片想要表达的情绪
4. **肤色优先保护**：无论分级方案多么大胆，肤色区域必须保持自然可识别——否则人看起来会像外星人

- Color Grading Central：https://www.colorgradingcentral.com/
- Lightroom色彩分级官方指南：https://helpx.adobe.com/lightroom-cc/how-to/color-grading.html

## 6.6 人像摄影常见后期错误与避免方法

![常见后期错误示例](images/editing/16.jpg)

即使是经验丰富的摄影师，也难免犯一些后期处理中的经典错误。以下是十个最常见的错误及其修正方法，值得定期对照自查。

**错误一：过度磨皮——"陶瓷人"效应**

**表现：** 皮肤完全失去纹理，像3D渲染的陶瓷人偶，面部没有任何毛孔、细纹或自然的皮肤肌理。

**原因：** 使用了过高的磨皮强度（Portraiture 平滑度 > 30，或美图秀秀默认磨皮），或频率分离中高频层被过度模糊。

**修正方法：** 
- 频率分离中高频层只修明显瑕疵，不处理整体纹理
- 磨皮插件平滑度控制在 10-20
- 最终叠加一层低不透明度的原图纹理
- **自查方法：** 放大到 200% 查看皮肤，如果看不到任何纹理，说明过度了

**错误二：过度液化——"蛇精脸"效应**

**表现：** 下巴尖到不自然、眼睛大到像动漫人物、脸型完全失去骨骼结构感、额头饱满到膨胀。

**原因：** 液化参数过于激进，或在手机APP中使用了一键瘦脸/大眼的高强度模式。

**修正方法：**
- 脸型液化不超过原宽度的 3-5%
- 大眼不超过 107%
- 下巴适当收窄但保持自然的骨骼转折
- **自查方法：** 将照片缩小到手机屏幕大小查看——如果看着不像真人，就需要回调

**错误三：白色色阶溢出——"死白"**

**表现：** 面部高光区域（额头、鼻尖、肩膀）完全没有细节，变成纯白色的"死白"。

**原因：** 曝光或白色色阶推得过高，高光恢复不够。

**修正方法：**
- 按住 Alt 拖动白色色阶，确保没有大面积蓝色警告
- 直方图右端不应该有大量堆积
- 如果高光已经溢出且无法恢复，使用阴影/高光工具或D&B技法手动重建

**错误四：暗部死黑**

**表现：** 头发、眼窝、鼻翼阴影等暗部完全没有细节，变成纯黑色。

**原因：** 黑色色阶压得过度，或对比度/去朦胧参数过高。

**修正方法：**
- 黑色色阶不要低于 -30
- 阴影适当提亮 +20 到 +40
- 按住 Alt 拖动黑色色阶检查

**错误五：色调不统一**

**表现：** 同一组照片中，有的偏黄、有的偏蓝、有的偏绿，放在一起观看时色彩跳跃。

**原因：** 自动白平衡不一致，或风格预设没有统一应用。

**修正方法：** 参照6.4节"肤色统一技术"，使用参考视图确保每张照片的色温和色调一致

**错误六：锐化过度**

**表现：** 皮肤毛孔被锐化得像钉子一样清晰，头发出现白色光晕（锐化晕轮），整体画面显得"硬"而不舒服。

**原因：** 全局锐化参数过高，没有区分需要锐化的区域（眼睛、头发）和不需要锐化的区域（皮肤）。

**修正方法：**
- 全局锐化数量控制在 40-70，半径 1.0
- 使用"蒙版"滑块（按住 Alt 拖动），让锐化只作用于边缘和轮廓，避开皮肤
- 或者导出后在PS中使用高反差保留+蒙版做局部锐化

**错误七：饱和度过高**

**表现：** 肤色橙红到发亮、衣服颜色鲜艳刺眼、整体画面像被泼了一层油漆。

**原因：** 自然饱和度和饱和度都推得过高，或色彩分级饱和度太高。

**修正方法：**
- 饱和度不要超过 +30，自然饱和度不要超过 +20
- 使用HSL面板单独控制每个颜色通道
- 如果需要鲜艳效果，优先使用"鲜艳度（Vibrance）"而非"饱和度（Saturation）"——前者保护已经饱和的颜色，主要提亮不饱和区域

**错误八：暗角过重**

**表现：** 画面四角严重变暗，像通过管子拍摄的一样。

**原因：** 后期暗角数量推得过高（-40以上），或叠加了镜头本身的暗角。

**修正方法：**
- 暗角数量控制在 -15 到 -30 之间
- 如果是镜头本身暗角，先在"镜头校正"面板中勾选"启用镜头配置文件"自动修正
- 暗角的目的是引导视线到画面中心，不是制造隧道效果

**错误九：忽略镜头畸变矫正**

**表现：** 人脸边缘变形（用广角镜头近距离拍摄时最明显）、建筑线条弯曲。

**原因：** 没有启用镜头校正配置文件，或拍摄距离太近导致透视变形。

**修正方法：**
- Lightroom中勾选"启用镜头配置文件"——这会自动识别镜头型号并应用矫正
- 对于透视变形，使用"变换"面板中的"自动"或"垂直"矫正
- 广角人像拍摄建议在 85mm 以上焦距进行，从根本上避免变形

**错误十：导出设置不当**

**表现：** 分辨率过低导致细节模糊，或文件过大导致加载缓慢，颜色配置错误导致色彩偏差。

**修正方法：** 参照6.8节的导出设置规范

**自查清单（每次出片前过一遍）：**
- [ ] 放大到100%检查皮肤纹理是否保留
- [ ] 缩小到实际展示尺寸检查液化是否自然
- [ ] 检查直方图两端是否有溢出
- [ ] 检查同组照片白平衡和风格是否一致
- [ ] 检查边缘是否有锐化晕轮
- [ ] 检查肤色是否自然可识别
- [ ] 在不同设备上预览（手机、电脑、不同亮度环境）

## 6.7 批量处理工作流

![批量处理工作流示意图](images/editing/17.jpg)

商业人像摄影中，一次拍摄可能产生 50-200 张甚至更多的照片。如果每张照片都从头调色精修，工作效率低到不可接受。批量处理工作流的目标是在保证质量的前提下，将单张照片的平均后期时间从 30-60 分钟压缩到 5-10 分钟。

**Lightroom批量处理流程：**

**第一步：筛选（Culling）**
1. 导入所有照片后，切换到"网格视图"（快捷键 G）
2. 使用标记（P）和排除（X）快速筛选：标记保留的照片，排除明显失败的照片（闭眼、严重脱焦、表情不佳）
3. 通常保留率在 30-50% 之间——100张原片中选出30-50张进入后期流程
4. 可以使用" surveyed"标记进一步区分：选一次标记，再选一次标记五星——将五星照片作为最终交付候选

**第二步：基准片调色**
1. 从候选照片中选出一张最具代表性的照片作为基准片——光线、角度、构图最典型的
2. 对基准片进行完整的调色流程：白平衡 → 基础调整 → HSL → 色调曲线 → 分级 → 局部调整
3. 调色过程中使用"虚拟副本"（Ctrl/Cmd + '）保存不同版本，方便对比选择

**第三步：同步设置**
1. 调好基准片后，按住 Ctrl/Cmd 点击所有需要应用同样参数的照片
2. 点击右侧面板底部的"同步"按钮
3. 在弹出的同步设置对话框中，**有选择地勾选**需要同步的项目：
   - ✅ 始终同步：处理版本、镜头矫正、色差去除、色调曲线、HSL、分离色调/色彩分级、颗粒、暗角
   - ⚠️ 谨慎同步：白平衡、曝光、对比度（每张照片的拍摄条件可能略有不同）
   - ❌ 不建议同步：裁剪、旋转、局部调整画笔、渐变滤镜、径向滤镜（这些是画面位置相关的，不同照片的构图不同）
4. 理想方案：先同步所有全局参数，然后逐张微调白平衡和曝光

**第四步：逐张微调**
1. 同步后，快速浏览每张照片
2. 微调项目：白平衡（±100K以内）、曝光（±0.3EV以内）、裁剪（如需要）
3. 每张照片微调时间控制在 1-2 分钟

**第五步：批量导出**
1. 全选所有完成调色的照片
2. 设置导出参数（参照6.8节）
3. 一键导出

**预设驱动的批量处理策略：**

如果你经常拍摄同类型的人像（如婚礼跟拍、证件照、写真），应该建立完整的预设链：

1. **基础修正预设**：自动白平衡 + 镜头矫正 + 去色差
2. **场景预设**（3-5个）：室内暖光、室内冷光、户外日光、户外阴天、混合光源
3. **风格预设**（2-3个）：你的标志性调色风格，按强度分为轻/中/重
4. **导出预设**：不同用途的导出设置（社交媒体、打印、客户交付）

**Photoshop批处理动作（Actions）：**

对于需要在PS中精修的照片，可以使用动作（Action）录制常用操作序列：

1. 打开"窗口 > 动作"面板
2. 点击"新建动作"按钮，开始录制
3. 依次执行：复制图层 → 高斯模糊（5px）→ 应用图像（减去）→ 线性光 → 频率分离基础设置
4. 停止录制
5. 使用"文件 > 自动 > 批处理"，对文件夹中所有照片执行该动作
6. 可以配合"Droplet"功能，将文件夹拖放到桌面图标上自动执行

**AI辅助批量处理的未来：**

2024-2025年，AI修图工具已经可以显著加速批量处理：
- **Luminar Neo**：AI自动识别人脸、皮肤、天空，一键应用针对性调整
- **Retouch4Me**：AI自动检测并修除皮肤瑕疵、美白牙齿、平滑皮肤，效果远超传统自动化
- **Adobe Firefly**：集成在Lightroom/PS中的AI功能，一键去除、生成式填充、AI蒙版
- 建议将AI工具用于基础修饰（磨皮、去瑕疵、白平衡校正），手动处理风格化和创意部分

**效率优化建议：**
- 键盘快捷键记忆10个最常用的操作，可以节省30%的操作时间
- 使用鼠标宏（如Logitech G系列）录制常用操作组合
- 双显示器工作流：一个显示器全屏预览照片，另一个显示调整面板
- 使用预设收藏夹组织常用预设，减少寻找预设的时间
- 对于重复性极高的工作（如证件照），考虑使用Python + OpenCV自动化基础调整

- Lightroom批量处理官方指南：https://helpx.adobe.com/lightroom-cc/how-to/batch-edit-photos.html
- SLR Lounge效率工作流：https://www.slrlounge.com/workflow

## 6.8 导出与分享

![不同平台的图片展示效果](images/editing/18.jpg)

精心调色和精修的照片，如果导出设置不当，可能在社交平台上显示效果大打折扣。本节详细讲解各平台的最佳导出设置和输出规范。

### 社交平台最佳尺寸

**微信朋友圈：**
- 最佳尺寸：1080 × 1080px（正方形）或 1080 × 1350px（4:5竖版）
- 格式：JPEG
- 品质：85-90%
- 色彩空间：sRGB（微信不支持P3/Adobe RGB）
- 文件大小：单张不超过 5MB（超过会被压缩，质量下降）
- 注意：微信会对上传图片进行二次压缩，即使你上传高质量原图也会被压缩到约 200-300KB。可以通过先缩小到 1200px 宽、品质 95% 的方式，让微信的二次压缩对画质的影响最小化

**小红书：**
- 最佳尺寸：1080 × 1440px（3:4竖版）
- 格式：JPEG
- 品质：90-95%
- 色彩空间：sRGB
- 文件大小：单张不超过 20MB
- 注意：小红书竖版图片展示效果远好于横版——竖版占满屏幕宽度，横版只有一半宽度。如果是人像写真，强烈建议裁剪为竖版
- 封面图建议：使用最吸引眼球的一张作为封面，裁剪为 3:4，并在图片1/3处留出文字空间（小红书封面通常叠加标题文字）

**Instagram：**
- Feed帖子：1080 × 1080px（正方形）或 1080 × 1350px（4:5竖版）
- Stories/Reels：1080 × 1920px（9:16竖版）
- 格式：JPEG（Feed帖子）或 MP4（Reels）
- 品质：90-95%
- 色彩空间：sRGB
- 注意：Instagram也会压缩图片，但品质控制比微信好很多。上传 1080px 宽度的图片基本不会被二次缩放

**微博：**
- 最佳尺寸：不超过 2048px 的长边
- 格式：JPEG
- 品质：90%
- 注意：微博对长图有特殊处理——宽度小于 1000px 的高度不限；宽度大于 1000px 时高度不能超过宽度的 3 倍

**通用社交媒体导出设置（Lightroom）：**
1. 长边：2048px（保留足够的清晰度，各平台都会自动缩放）
2. 分辨率：72 ppi（屏幕显示，不需要300ppi）
3. 品质：90%
4. 色彩空间：sRGB
5. 锐化：选择"屏幕"，标准
6. 文件名：`日期_序号_平台.jpg`，如 `20260404_01_INS.jpg`

### 打印输出设置

![打印输出参数对照表](images/editing/19.jpg)

打印输出和屏幕显示的参数要求完全不同——打印需要更高的分辨率、更大的色彩空间和锐化策略。

**分辨率要求：**
- 标准打印：300 ppi（最低 240 ppi，不推荐低于此值）
- 大幅面打印（海报、展板）：150-200 ppi（因为观看距离更远，分辨率要求可以降低）
- 计算方法：打印尺寸 × 300 = 所需像素。例如 8×10 英寸打印需要 2400 × 3000 像素

**色彩空间：**
- 家用喷墨打印机（Epson、Canon、HP）：Adobe RGB 或打印机自定义 ICC
- 商业冲印（淘宝冲印店）：sRGB（大多数冲印店只支持sRGB）
- 高端艺术微喷：Adobe RGB + 定制 ICC Profile
- 重要原则：如果不确定打印店的色彩空间支持情况，统一使用 sRGB——虽然色域较窄，但不会出现色彩偏移

**导出设置（Lightroom打印导出）：**
1. 分辨率：300 ppi
2. 色彩空间：根据打印方式选择（见上）
3. 锐化：选择" matte"（哑光纸）或"glossy"（光面纸），标准或高
4. 品质：100%（打印需要最高品质）
5. 格式：JPEG（冲印）或 TIFF（高端艺术微喷）

**常见打印尺寸与像素对应：**

| 打印尺寸（英寸） | 厘米（约） | 所需像素（300ppi） |
|-----------------|-----------|-------------------|
| 4×6 | 10×15cm | 1200×1800 |
| 5×7 | 13×18cm | 1500×2100 |
| 8×10 | 20×25cm | 2400×3000 |
| 8×12 | 20×30cm | 2400×3600 |
| 11×14 | 28×36cm | 3300×4200 |
| 16×20 | 40×50cm | 4800×6000 |
| 20×30 | 50×76cm | 6000×9000 |

**打印前的检查：**
1. 放大到 100% 检查是否有明显的修图痕迹（打印比屏幕更容易看到瑕疵）
2. 检查边缘是否有锐化晕轮
3. 确认裁剪比例正确——打印比例与屏幕比例不同，可能需要重新裁剪
4. 在屏幕上模拟打印效果：Photoshop中"视图 > 校样颜色"，选择目标配置文件

### 水印添加

水印是保护作品版权的基本手段，但必须在保护效果和画面美感之间找到平衡。一个太大太突兀的水印会毁掉整张照片的观感。

**水印设计原则：**
1. **位置**：右下角或左下角是最常见的选择，不会遮挡主体。避免将水印放在人物面部或画面中心
2. **大小**：高度约为图片高度的 3-5%。小到不影响观感，大到能防止被直接裁切
3. **颜色**：白色 + 50% 不透明度（深色背景图片）；黑色 + 50% 不透明度（浅色背景图片）；或者使用灰色（#808080）+ 40% 不透明度——灰色在大多数背景上都可见但不突兀
4. **内容**：摄影师名字/品牌名 + 网站/社交媒体账号。不要加水印以外的信息

**Lightroom水印设置：**
1. 编辑 > 编辑水印（Lightroom Classic）或 设置 > 水印（Lightroom CC）
2. 支持文字水印和图形水印（Logo图片）
3. 可以设置字体、大小、不透明度、位置、内边距
4. 建议创建两个水印预设：社交媒体版（稍大，含社交媒体账号）和打印版（较小，含名字和网站）

**Photoshop水印进阶技巧：**
1. 将水印放在独立图层，保存为PSD模板——每次只需要替换照片图层
2. 批量添加水印：使用"文件 > 自动 > 批处理"或Droplet
3. 可见水印 vs 不可见水印：在低分辨率网络版本使用可见水印，在高分辨率交付版本使用不可见的数字水印（如Photoshop的"滤镜 > Digimarc > 嵌入水印"）

**导出文件命名规范：**
```
项目名_日期_序号_版本_用途
例如：
婚礼写真_20260404_001_v2_INS.jpg
证件照_20260404_001_v1_PRINT.tif
```

**交付文件夹结构建议：**
```
项目名_交付/
├── RAW原文件/           （可选，根据合同）
├── 高清成片_sRGB/       （供客户留存，JPEG 100%，原始分辨率）
├── 网络版_水印/         （社交媒体使用，2048px，含水印）
├── 打印版_AdobeRGB/     （打印使用，TIFF，300ppi）
└── 选片联系人表.pdf      （可选）
```

## 本章小结

后期处理是人像摄影创作链中不可或缺的一环。本章从Lightroom的全局调色基础（曝光、白平衡、HSL肤色控制、预设系统）出发，深入到Photoshop的像素级精修（频率分离磨皮、液化塑形、眼睛精修），再到手机APP的快速修图方案，以及肤色统一、色彩分级、批量处理工作流和导出规范。

核心要点回顾：
1. **先校正后创意**：白平衡和曝光校正永远是第一步，风格化调色建立在中性基底之上
2. **保留自然纹理**：皮肤纹理是人像真实感的根基，磨皮以"干净"为目标，不以"光滑"为目标
3. **克制是高级感的来源**：液化、饱和度、锐化、暗角——所有参数的"高级"区间都在中间偏低的范围
4. **效率通过系统化实现**：预设链、批量处理流程、键盘快捷键——投入一次时间建设效率系统，长期回报巨大
5. **输出适配场景**：不同平台有不同的最优尺寸和品质要求，导出前确认目标平台规范
